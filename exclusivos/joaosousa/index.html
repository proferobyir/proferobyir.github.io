# =======================================================
# crud_ops.py - OPERACIONES CRUD
# =======================================================

from db_config import get_connection

# =======================================================
# CREATE - Insertar un nuevo usuario
# =======================================================
def crear_usuario(nombre, email):
    conn = get_connection()
    cursor = conn.cursor()
    sql = "INSERT INTO usuarios (nombre, email) VALUES (%s, %s)"
    cursor.execute(sql, (nombre, email))
    conn.commit()
    conn.close()

# =======================================================
# READ - Obtener todos los usuarios
# =======================================================
def obtener_usuarios():
    conn = get_connection()
    cursor = conn.cursor(dictionary=True)
    cursor.execute("SELECT * FROM usuarios")
    resultados = cursor.fetchall()
    conn.close()
    return resultados

# =======================================================
# UPDATE - Actualizar un usuario existente
# =======================================================
def actualizar_usuario(id, nombre, email):
    conn = get_connection()
    cursor = conn.cursor()
    sql = "UPDATE usuarios SET nombre=%s, email=%s WHERE id=%s"
    cursor.execute(sql, (nombre, email, id))
    conn.commit()
    conn.close()

# =======================================================
# DELETE - Eliminar un usuario por ID
# =======================================================
def eliminar_usuario(id):
    conn = get_connection()
    cursor = conn.cursor()
    cursor.execute("DELETE FROM usuarios WHERE id=%s", (id,))
    conn.commit()
    conn.close()

🧭 El viaje de los datos (explicado)

1. Usuario escribe en Entry → entry_nombre.get() captura el texto.
2. Botón "Guardar" → ejecuta guardar_datos() (en main.py).
3. guardar_datos() → llama a crear_usuario(nombre, email) (desde crud_ops.py).
4. crud_ops.crear_usuario() → llama a get_connection() (desde db_config.py).
5. get_connection() → retorna un objeto conn (MySQLConnection).
6. Sobre conn, creamos cursor → cursor = conn.cursor().
7. Ejecutamos SQL → cursor.execute("INSERT ...", (nombre, email)).
8. conn.commit() → confirma la transacción en la BD.
9. conn.close() → liberamos la conexión.
10. Mensaje de éxito → el usuario ve confirmación y la lista se actualiza.

✅ Buenas prácticas en Windows

  • Usa entorno virtual → evita conflictos de paquetes.
  • Separa responsabilidades → interfaz (main.py) / lógica (crud_ops.py) / conexión (db_config.py).
  • Maneja excepciones → try/except en operaciones de base de datos.
  • Cierra siempre la conexión → conn.close() para evitar fugas.
  • Usa parametrización → (%s) para evitar inyección SQL.
  • Variables globales → decláralas al inicio del archivo para que todas las funciones las vean.
