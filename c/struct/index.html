🧠 ¿Qué es struct?

Una estructura permite agrupar variables de diferentes tipos (int, float, char, etc.) en un solo bloque lógico. Ideal para representar entidades del mundo real como alumnos, libros, puntos, fechas.

// Definición de un struct Persona
struct Persona {
    char nombre[50];
    int edad;
    float altura;
};
🔹 Sintaxis: struct Nombre { miembros };
🔹 Después puedes declarar variables: struct Persona p1;
🔹 Acceso a miembros: p1.edad = 21; usando el operador punto .
// Declaración + inicialización
struct Fecha {
  int dia, mes, anio;
};
struct Fecha hoy = {25, 12, 2025};
// Acceso y asignación
printf("Día: %d", hoy.dia);
hoy.mes = 1;
// También arrays de structs
✅ Ventajas: Organiza información relacionada, facilita paso de parámetros a funciones, modela objetos simples en C.

⚙️ Ejemplo completo + typedef

typedef struct {
   char nombre[30];
   int codigo;
   float nota;
} Estudiante;

int main() {
   Estudiante e1 = {"Ana", 101, 9.5};
   printf("%s - nota: %.1f", e1.nombre, e1.nota);
   return 0;
}
📌 typedef permite evitar escribir struct cada vez.
📌 Los miembros pueden ser cualquier tipo, incluso otros structs (anidación).
📌 Para punteros a struct se usa -> : ptr->edad = 22;
🎯 Ejemplo con puntero a struct:
Estudiante *ptr = &e1;
printf("%s", ptr->nombre);
💡 Las estructuras se pasan por valor a funciones, pero si son grandes es mejor usar punteros por eficiencia.
