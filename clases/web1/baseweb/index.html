🌐 ¿Qué es HTML? Paso 1

HTML (HyperText Markup Language) es el lenguaje de marcado que usamos para estructurar el contenido de las páginas web. Piensa en HTML como los huesos de un sitio web: define dónde va cada cosa.

<h1>¡Hola, mundo!</h1>  →  Esto crea un título grande

📌 ¿Cómo funciona? HTML usa etiquetas (también llamadas tags) que son como instrucciones para el navegador. La mayoría de las etiquetas vienen en pares:

<etiqueta> contenido </etiqueta>  →  Apertura y cierre
💡

Tip para principiantes: HTML no es un lenguaje de programación, es un lenguaje de marcado. No hace cálculos ni toma decisiones, solo organiza el contenido.

Las etiquetas más comunes que usarás hoy:

  • <h1> a <h6> → Títulos (del más importante al menos importante)
  • <p> → Párrafos de texto
  • <ul> + <li> → Listas con viñetas
  • <div> → Un "cajón" para agrupar contenido
  • <img src="rutadearchivodeimagen" / > → Para mostrar imágenes
  • <video> → Para reproducir videos
  • <iframe> → Para mostrar contenido de otras páginas

🎯 Tu objetivo: Entender que cada etiqueta tiene un propósito y que combinándolas puedes crear cualquier página web.

🏗️ Estructura Básica de una Página HTML Paso 1.5

Toda página HTML tiene una estructura fundamental que es como el esqueleto del documento. A continuación, te mostramos sus partes esenciales:

<!-- 1. Declaración del tipo de documento --> <!DOCTYPE html> <!-- 2. Raíz del documento: <html> --> <html lang="es"> <!-- 3. Cabecera: metadatos, título, estilos --> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Mi Página Web</title> <style> /* CSS aquí */ </style> </head> <!-- 4. Cuerpo: contenido visible --> <body> <h1>¡Hola, mundo!</h1> <p>Este es mi primer párrafo.</p> </body> </html>

1. <!DOCTYPE html>

Declaración del tipo de documento. Le dice al navegador que es un documento HTML5. ¡Siempre debe ir primero!

2. <html lang="es">

Raíz del documento. Contiene todo el código HTML. El atributo lang indica el idioma de la página.

3. <head>

Cabecera o "cerebro" de la página. Aquí va información no visible para el usuario: metadatos (como el charset), el título de la pestaña, los estilos CSS y más.

4. <body>

El cuerpo o "cara visible" de la página. Todo lo que pongas aquí será lo que vean los usuarios: textos, imágenes, botones, etc.

🧠

Regla de oro: <html> envuelve todo, <head> es la configuración (invisible) y <body> es el contenido visible. ¡Así de simple!

🎯 Tu objetivo: Identificar estas cuatro partes en cualquier página web que visites. ¡Es el primer paso para entender cómo se construye la web!

🎨 ¿Qué es CSS? Paso 2

CSS (Cascading Style Sheets) es el lenguaje que embellece nuestras páginas web. Si HTML es el esqueleto, CSS es la ropa, el maquillaje y la personalidad del sitio.

h1 { color: gold; font-size: 3rem; }  →  Todos los títulos h1 serán dorados y grandes

¿Qué puedes hacer con CSS?

  • 🎨 Colores: Cambiar el color de fondo, texto, bordes
  • 📏 Tamaños: Controlar el ancho, alto, márgenes
  • 🖼️ Diseño: Posicionar elementos con Flexbox o Grid
  • ✨ Animaciones: Hacer que los elementos se muevan, roten o cambien
  • 📦 Variables CSS: Guardar valores para reutilizarlos (¡lo veremos ahora!)
💡

Tip: El CSS se escribe dentro de la etiqueta <style> en el <head> de tu HTML, o en un archivo aparte con extensión .css. ¡Así separamos el contenido (HTML) del estilo (CSS)!

📦 Variables en CSS (Custom Properties) Paso 2.5

Las variables en CSS (también llamadas Custom Properties) son como contenedores donde guardamos valores que queremos reutilizar en todo nuestro código. ¡Son una herramienta poderosísima que te ahorrará mucho tiempo!

:root { --color-principal: #ffd700; --espacio-grande: 30px; }  →  Declaramos variables en :root (accesibles en toda la página)
💡

Analogía: Piensa en las variables CSS como etiquetas en frascos. Si pones "color dorado" en un frasco llamado --color-dorado, puedes usar ese color en muchos lugares cambiando un solo frasco.

📌 ¿Cómo se usan?

📝 1. Declarar una variable

Se declaran con dos guiones -- y un nombre:

--mi-color: #ff6b6b;

Siempre dentro de un selector (como :root)

🔄 2. Usar una variable

Se usan con la función var():

color: var(--mi-color);

Así el navegador sabe que debe usar el valor guardado

🎯 3. ¿Dónde se declaran?

En :root (global) o en selectores específicos (local):

:root { --global: #ffd700; }
.tarjeta { --local: #4ecdc4; }

🔁 4. Ventaja principal

Cambias el valor una vez y se actualiza en todos lados:

:root { --color: red; }
/* Cambias a blue y todo cambia */

🎯 Beneficio para ti: Si tienes un sitio con 50 lugares donde usas el mismo color, con variables solo cambias una línea y todos los colores se actualizan. ¡Imagina el tiempo que ahorras!

🖼️ ¿Qué es un <iframe>? Paso 3

Un iframe (Inline Frame) es como una ventana dentro de otra ventana. Te permite mostrar contenido de otras páginas dentro de la tuya. Es súper útil para:

  • 📍 Mapas interactivos (Google Maps)
  • 📹 Videos de YouTube
  • 📊 Gráficos de otras plataformas
  • 📰 Contenido de terceros
<iframe src="https://www.google.com/maps/..." width="100%" height="300"></iframe>
🔒

Importante: No todas las páginas se pueden mostrar en un iframe. Algunos sitios tienen políticas de seguridad que lo bloquean (como bancos o redes sociales). ¡Pero muchas páginas como mapas o videos sí lo permiten!
