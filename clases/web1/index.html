Tu guía rápida con conceptos y plantilla lista para usar

1. Conceptos Básicos esenciales

<!DOCTYPE html>: Declaración obligatoria al inicio del archivo. Le dice al navegador que estás usando la versión moderna de HTML (HTML5).
<html>: La etiqueta raíz que envuelve absolutamente todo el documento. El atributo lang="es" le indica el idioma de la página a los buscadores.
<head>: La sección del "cerebro" de la página. Contiene información técnica para el navegador y los buscadores (metadatos, enlaces a hojas de estilo CSS), pero nada de esto se ve directamente en la pantalla.
<body>: El "cuerpo" de la página. Aquí se coloca todo lo que el usuario final va a ver, como textos, imágenes, videos, formularios y menús.

2. Plantilla HTML5 Estándar

Puedes copiar el siguiente fragmento directamente en Visual Studio Code para iniciar cualquier proyecto web desde cero:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Nombre de tu Proyecto</title>
</head>
<body>

    <h1>¡Hola Mundo!</h1>
    <p>Tu contenido comienza aquí.</p>

</body>
</html>
