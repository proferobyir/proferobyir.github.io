1. Nuevas Etiquetas Explicadas (SEO e Íconos)

<meta name="description" ...>: Es el texto resumen que aparece debajo del título de tu página cuando alguien la busca en Google. Es crucial para el SEO.
<meta name="keywords" ...>: Una lista de palabras clave relacionadas con tu página (aunque los buscadores modernos le dan menos peso hoy en día, sigue siendo buena práctica).
<link rel="icon" ...>: Define el **Favicon**. Es la etiqueta encargada de colocar el logo en la pestaña del navegador y es exactamente la misma que escanea Google para mostrar el ícono al lado del nombre de tu web en los celulares y computadoras.

2. Plantilla HTML5 Completa (Con SEO y Favicon)

Puedes copiar esta estructura base ideal para iniciar cualquier proyecto profesional:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    
    <!-- SEO Básico -->
    <meta name="description" content="Descripción corta de tu página web (máximo 160 caracteres).">
    <meta name="keywords" content="palabra1, palabra2, palabra3">
    <meta name="author" content="Tu Nombre o Empresa">
    
    <!-- Ícono de la pestaña y buscadores -->
    <link rel="icon" href="favicon.png" type="image/png" sizes="32x32">
    
    <title>Título de la Página</title>
</head>
<body>

    <h1>Mi Sitio Web Profesional</h1>
    <p>El código está optimizado para motores de búsqueda.</p>

</body>
</html>

💡 Regla de oro de Google para mostrar tu ícono:

Para que Google muestre el ícono de tu web en sus resultados de búsqueda, el archivo (por ejemplo, favicon.png) debe estar guardado en la **carpeta raíz** de tu proyecto (junto al archivo index.html) y tu sitio debe estar publicado en internet. Google no mostrará el ícono si estás abriendo el archivo de forma local (file:///C:/...).

Estructura optimizada para la enseñanza de desarrollo web.
