Guía metodológica interactiva con ejemplos de listas y pilares de la web.

Glosario de Términos Fundamentales

HTML Estructura

HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el esqueleto o la base estructural de un sitio web. Define el contenido básico a través de etiquetas (textos, imágenes, contenedores). **No es un lenguaje de programación**.

CSS Diseño

Cascading Style Sheets (Hojas de Estilo en Cascada). Es el lenguaje encargado del diseño visual, los colores, fuentes, espaciados y adaptabilidad (responsividad). Controla cómo se muestran estéticamente los elementos HTML.

JavaScript Interactividad

Es un lenguaje de programación real interpretado por el navegador. Se encarga de aportar dinamismo, lógica y comportamiento interactivo a la página (animaciones, validaciones de formularios, carga de datos en tiempo real).

1. La Analogía del Cuerpo Humano

Estrategia visual para explicar las etiquetas de nivel superior:

<html>

El Cuerpo Completo: Envuelve y encierra absolutamente todo el código del documento.

<head>

La Mente: Datos de configuración e instrucciones lógicas que el navegador interpreta pero el usuario no ve.

<body>

La Parte Visible: Contiene todo lo que se renderiza directamente en pantalla para la interacción.

2. Código Base de la Plantilla

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página Web</title>



    <h1>¡Hola, mundo!</h1>
    <p>Este es el inicio de mi camino en el desarrollo web.</p>

</body>
</html>

3. Desglose Técnico

  • <!DOCTYPE html> Activa el modo estándar de renderizado HTML5.
  • lang="es" Ayuda a los motores de búsqueda y lectores asistidos.
  • charset="UTF-8" Asegura que los acentos y la "ñ" no se rompan en servidores remotos.
  • name="viewport" Sincroniza la escala visual con el ancho físico del dispositivo móvil.

4. Ejemplos Prácticos de Listas

Muestre a sus estudiantes la diferencia entre agrupar información con o sin orden jerárquico:

Listas Desordenadas (<ul>)

Usa viñetas genéricas cuando el orden de los elementos no altera el sentido.

  • Computadora de escritorio o laptop
  • Editor de código (ej. VS Code)
  • Navegador web moderno (Chrome, Firefox, Edge)
  • Ganas de experimentar y aprender
<ul>
  <li>Computadora</li>
  <li>Editor de código</li>
</ul>

Listas Ordenadas (<ol>)

Usa números correlativos cuando sea un procedimiento secuencial paso a paso.

  1. Crear un archivo nuevo de texto en la computadora.
  2. Guardar el archivo con el nombre exacto index.html.
  3. Escribir o pegar la estructura base de las etiquetas.
  4. Hacer doble clic sobre el archivo para abrirlo en el navegador.
<ol>
  <li>Crear archivo</li>
  <li>Abrir en navegador</li>
</ol>

5. Dinámicas Sugeridas para el Aula

  1. El Reto del Título: Modificar el contenido de <title> y comprobar su visualización en la pestaña del navegador.
  2. Desafío de las Listas: Pídales que añadan una sección propia dentro del ` construyendo una lista desordenada con sus canciones favoritas y una lista ordenada con los pasos de su receta de cocina preferida.
