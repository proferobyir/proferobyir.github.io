Clase 1: Estructura Inicial HTML5

Guía metodológica interactiva para comenzar el diseño y desarrollo web.

1. La Analogía del Cuerpo Humano

Ideal para explicar la jerarquía del DOM a los alumnos de manera intuitiva:

<html>

El Cuerpo Completo: Delimita y almacena todo el código fuente de la página web.

<head>

La Mente: Contiene metadatos, configuraciones y enlaces del sistema. Es lo que el navegador procesa internamente pero no dibuja.

<body>

El Cuerpo Visible: Todo lo que el usuario final interactúa: textos, imágenes, enlaces y componentes.

2. Código Base Estructurado

Copia y pega este bloque limpio en el editor para iniciar:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página Web</title>
</head>
<body>

    <h1>¡Hola, mundo!</h1>
    <p>Este es el inicio de mi camino en el desarrollo web.</p>

</body>
</html>

3. Desglose Técnico

  • <!DOCTYPE html> Informa al motor del navegador que interprete el código según el estándar HTML5 moderno.
  • lang="es" Atributo clave de accesibilidad que define el idioma nativo de la página.
  • charset="UTF-8" Tabla de codificación de caracteres que permite acentos, la "ñ" y caracteres especiales.
  • name="viewport" Declaración obligatoria para el diseño responsivo en móviles.
  • <title> Controla el texto expuesto en las pestañas del software de navegación.

4. Dinámicas Prácticas para el Aula

  1. El Reto del Título: Haga que los alumnos modifiquen el contenido interno del tag <title> y vean el cambio instantáneo en la barra del navegador.
  2. Simulación de Error: Pídales borrar el metadato charset="UTF-8" e insertar textos complejos con acentos o eñes (Ej. "Mañana, Computación") para observar cómo se rompe la codificación gráfica del texto.
