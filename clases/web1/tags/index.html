1. Etiquetas de Título (h1 - h6)

Jerarquía de encabezados para estructurar tu contenido

¿Qué son? Las etiquetas <h1> hasta <h6> definen títulos y subtítulos. <h1> es el más importante (título principal) y <h6> el menos importante.
titulos.html
✓ Listo para editar
Vista Previa
💡 Consejos:
  • Usa solo un <h1> por página (mejor para SEO)
  • Mantén el orden jerárquico (no saltes de h1 a h4)
  • Los títulos ayudan a los lectores de pantalla
L

2. Listas (ul y ol)

Listas desordenadas y ordenadas

Tipos de listas: <ul> (unordered list) crea listas con viñetas (puntos). <ol> (ordered list) crea listas numeradas. Ambas usan <li> para cada elemento.
listas.html
✓ Listo para editar
Vista Previa
💡 Consejos:
  • Usa <ul> cuando el orden no importe
  • Usa <ol> para pasos, rankings o secuencias
  • Puedes anidar listas (una lista dentro de otra)
I

3. Imágenes (img)

Incorpora imágenes en tu página web

Etiqueta img: <img> es una etiqueta vacía (no tiene cierre). Requiere src (ruta de la imagen) y alt (texto alternativo para accesibilidad).
imagenes.html
✓ Listo para editar
Vista Previa
💡 Consejos:
  • Siempre incluye el atributo alt (accesibilidad y SEO)
  • Usa max-width: 100% para que sean responsive
  • Optimiza el tamaño de las imágenes para web
  • Formatos recomendados: JPG (fotos), PNG (transparencia), SVG (vectores)
V

4. Video (video)

Reproduce videos directamente en tu página

Etiqueta video: <video> permite incrustar videos. Usa el atributo controls para mostrar los controles de reproducción. Puedes incluir múltiples fuentes con <source>.
video.html
✓ Listo para editar
Vista Previa
💡 Consejos:
  • Siempre incluye controls para que el usuario pueda controlar el video
  • Proporciona múltiples formatos (MP4 y WebM/OGG) para compatibilidad
  • Usa poster para mostrar una imagen antes de reproducir
  • Atributos útiles: autoplay, loop, muted
T

5. Input de Texto (input type="text")

Campos de entrada de texto para formularios

Input text: <input type="text"> crea un campo de una línea para texto. Usa placeholder para texto de ayuda, required para hacerlo obligatorio, y maxlength para limitar caracteres.
input-text.html
✓ Listo para editar
Vista Previa
💡 Consejos:
  • Usa <label> siempre asociado al input (accesibilidad)
  • placeholder no reemplaza al label, es solo una guía
  • Para emails, mejor usa type="email"
  • autocomplete ayuda a los usuarios a completar datos
C

6. Input de Color (input type="color")

Selector de color nativo del navegador

Input color: <input type="color"> abre un selector de color nativo. El valor siempre es un código hexadecimal (#RRGGBB). Ideal para personalizaciones de tema o preferencias de color.
input-color.html
✓ Listo para editar
Vista Previa
💡 Consejos:
  • El valor por defecto debe ser un hex válido (#RRGGBB)
  • No todos los navegadores lo soportan igual (verificar compatibilidad)
  • Puedes usar JavaScript para leer el valor seleccionado
  • Ideal para personalización de temas o preferencias
D

7. Divisores (div)

Contenedores genéricos para agrupar contenido

Etiqueta div: <div> es un contenedor de bloque genérico. No tiene significado semántico por sí solo, pero es fundamental para organizar el layout y aplicar estilos CSS o JavaScript a grupos de elementos.
div.html
✓ Listo para editar
Vista Previa
💡 Consejos:
  • Usa divs para agrupar elementos relacionados
  • No abuses de los divs - usa etiquetas semánticas cuando puedas (header, section, article)
  • Los divs son bloques por defecto (ocupan todo el ancho)
  • Fundamentales para CSS Grid y Flexbox
B

8. Botones (button)

Elementos interactivos para acciones

Etiqueta button: <button> crea un botón clickeable. Puede contener texto, iconos o HTML. Usa type="submit" en formularios o type="button" para acciones JavaScript.
botones.html
✓ Listo para editar
Vista Previa
💡 Consejos:
  • Siempre especifica type (button, submit, reset)
  • Usa disabled para deshabilitar botones
  • Los botones pueden contener HTML (iconos, imágenes)
  • Importante para accesibilidad: texto claro y descriptivo
A

9. Enlaces Internos (a href="#")

Navegación dentro de la misma página

Anclas internas: <a href="#id"> crea enlaces que saltan a secciones dentro de la misma página. El destino debe tener id="nombre" coincidente. Esencial para navegación en páginas largas (índice, "volver arriba", etc.).
enlaces.html
✓ Listo para editar
Vista Previa (usa el menú)
💡 Consejos:
  • Usa html { scroll-behavior: smooth; } para scroll animado
  • El id debe ser único en la página
  • Perfecto para crear tablas de contenido
  • Botón "volver arriba" mejora la UX en páginas largas
