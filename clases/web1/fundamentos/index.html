Introducción a HTML y CSS

Primeros pasos en el desarrollo web - Desde cero

¿Qué es HTML?

HTML (HyperText Markup Language) es el esqueleto de toda página web. Define la estructura: títulos, párrafos, imágenes, enlaces...

Piensa en HTML como los ladrillos de una casa - sin ellos, no hay estructura.

Tu primera página web

Copia este código en un archivo llamado index.html y ábrelo en tu navegador:

<!DOCTYPE html>
<html>
<head>
    <title>Mi primera página</title>
</head>
<body>
    <h1>¡Hola Mundo!</h1>
    <p>Esta es mi primera página web.</p>
</body>
</html>

Etiquetas HTML básicas que debes conocer:

  • <h1> a <h6> → Títulos (h1 es el más importante)
  • <p> → Párrafos de texto
  • <a href="url"> → Enlaces
  • <img src="imagen.jpg" alt="descripción"> → Imágenes
  • <div> → Contenedor genérico
  • <ul> / <ol> → Listas

¿Qué es CSS?

CSS (Cascading Style Sheets) es el estilo de la página. Controla colores, tamaños, posiciones y fuentes.

Si HTML son los ladrillos, CSS es la pintura, los muebles y la decoración.

Ejemplo práctico: CSS básico

<style>
    h1 {
        color: blue;
        font-size: 40px;
        text-align: center;
    }
    p {
        color: gray;
        font-family: Arial;
    }
</style>

Propiedades CSS esenciales:

  • color → color del texto
  • background-color → color de fondo
  • font-size → tamaño de letra
  • margin / padding → espacios externos/internos
  • border → bordes
  • display: flex → layout flexible

Estructura de una página HTML

Toda página HTML tiene esta estructura básica:

<!DOCTYPE html>      ← Le dice al navegador que es HTML5
<html>               ← Etiqueta raíz
  <head>             ← Metadatos (no visibles)
    <title>Título</title>
    <link rel="stylesheet" href="estilos.css">
  </head>
  <body>             ← Todo lo visible va aquí
    <header>Cabecera</header>
    <main>Contenido principal</main>
    <footer>Pie de página</footer>
  </body>
</html>
Importante: Las etiquetas se abren y cierran. Ejemplo: <p>texto</p>

Modelos de caja: Flexbox y Grid

CSS tiene dos superpoderes para organizar elementos en la página:

Flexbox (una dimensión)

.contenedor {
    display: flex;
    justify-content: center;  /* horizontal */
    align-items: center;      /* vertical */
    gap: 20px;                /* espacio entre elementos */
}

Grid (dos dimensiones)

.contenedor {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;  /* 3 columnas iguales */
    gap: 15px;
}

Principios de diseño y color

Un buen diseño no es solo tecnología, también requiere reglas visuales:

  • Tipografía: Usa fuentes legibles (max 2-3 tipos)
  • Jerarquía visual: Los títulos más grandes → más importantes
  • Espaciado: El espacio en blanco mejora la lectura

¿Cómo funciona Internet?

Cuando abres una página web, ocurre esto:

Cliente → Servidor → Cliente: Tu navegador pide archivos a un servidor remoto y este los envía.

Accesibilidad web (a11y)

WCAG son las pautas para que todos puedan usar tu web, incluso personas con discapacidad.

Buenas prácticas básicas:
  • Siempre usa alt="descripción" en las imágenes
  • Contraste suficiente entre texto y fondo
  • Se puede navegar solo con el teclado (tecla TAB)

¿Cómo publicar tu página?

GitHub Pages es gratuito y fácil:

  1. Crea un repositorio en GitHub
  2. Sube tu archivo index.html
  3. Ve a Settings → Pages → Activa la rama main
  4. ¡Listo! Tu página estará en https://tusuario.github.io/repo/

¡Manos a la obra!

Tu primer desafío: Crea una tarjeta de presentación con HTML y CSS.

<div style="background: white; padding: 20px; border-radius: 12px; text-align: center;">
    <h2 style="color: #764ba2;">Tu Nombre</h2>
    <p>Desarrollador web en formación</p>
    <a href="#" style="color: #667eea;">Mi GitHub</a>
</div>

Objetivo: Modifica los colores, cambia el texto y agrega una imagen.
