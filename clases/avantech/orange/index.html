Tutorial Orange

Aprende Machine Learning sin código

¿Qué es Orange y por qué aprenderlo?

Orange es una plataforma visual de código abierto para minería de datos y machine learning. En lugar de escribir código, conectas componentes visuales llamados "widgets" para crear flujos de trabajo completos de análisis de datos.

¿Para qué sirve?

  • Analizar datos sin necesidad de programar en Python o R
  • Crear modelos predictivos de manera visual e intuitiva
  • Visualizar resultados con gráficos interactivos
  • Explorar patrones en tus datos rápidamente

En este tutorial aprenderás: desde cargar tus primeros datos hasta entrenar modelos avanzados, todo paso a paso con ejemplos prácticos.

Paso 1: Cargar y explorar datos

Principiante

1 ¿Qué vamos a hacer?

El primer paso en cualquier proyecto de machine learning es cargar tus datos. En Orange, esto se hace con el widget "File" o "Datasets".

📚 Ejemplo práctico: Vamos a usar el famoso dataset Iris, que contiene medidas de 150 flores de 3 especies diferentes (setosa, versicolor y virginica).

2 Widgets que necesitas

Herramientas básicas:

  • File: Carga archivos de datos (CSV, Excel, etc.)
  • Data Table: Muestra tus datos en formato tabla
  • Distributions: Visualiza la distribución de variables
  • Box Plot: Compara distribuciones entre grupos

3 Cómo conectar widgets

En Orange, conectas widgets arrastrando líneas entre ellos. El flujo de datos va de izquierda a derecha:

Flujo básico:
File → Data Table (para ver los datos)
File → Distributions (para analizar variables)
File → Box Plot (para comparar grupos)

Paso 2: Preprocesar datos

Intermedio

1 ¿Por qué preprocesar?

Los datos reales casi nunca están perfectos. Pueden tener:

  • Valores faltantes (celdas vacías)
  • Diferentes escalas (una variable va de 0-1, otra de 0-1000)
  • Columnas innecesarias que no aportan información

Preprocesar significa limpiar y preparar tus datos para que los modelos funcionen mejor.

2 Técnicas esenciales

Widgets de preprocesamiento:

  • Select Columns: Elige qué columnas usar
  • Impute: Rellena valores faltantes (con media, mediana, etc.)
  • Normalize: Escala variables a la misma escala
  • Feature Selection: Selecciona las variables más importantes

3 Ejemplo práctico

Problema: Imagina que tienes datos de casas con precios en dólares (100,000 - 1,000,000) y número de habitaciones (1 - 10). Las escalas son muy diferentes.

Solución: Usamos Normalize para que ambas variables tengan media 0 y desviación 1. Así el modelo no se confunde con los números grandes.

Paso 3: Entrenar modelos

Avanzado

1 ¿Qué es un modelo?

Un modelo de machine learning es un algoritmo que aprende patrones de tus datos para hacer predicciones. Por ejemplo: predecir si un email es spam o no, o clasificar flores por especie.

2 Tipos de modelos en Orange

Algoritmos disponibles:

  • Tree: Árbol de decisión (fácil de interpretar)
  • Random Forest: Múltiples árboles (más preciso)
  • SVM: Máquinas de vectores de soporte
  • Neural Network: Redes neuronales (avanzado)

3 Evaluar el modelo

Después de entrenar un modelo, necesitas saber qué tan bien funciona. Para esto usamos Test & Score, que divide tus datos en entrenamiento y prueba.

Métricas importantes:
• Accuracy: % de predicciones correctas (ej: 94% = muy bueno)
• AUC: Qué tan bien separa las clases (0.5 = azar, 1.0 = perfecto)
• F1-score: Balance entre precisión y recall
