📋 Cómo crear un Bot en Telegram y conectarlo a n8n

1
📱 Crear el Bot con BotFather en Telegram
Abre Telegram y busca @BotFather. Envía el comando /newbot. Sigue las instrucciones: elige un nombre y un username (debe terminar en 'bot'). Al final, BotFather te dará un Token HTTP API (ej: 00000000000000000000000000000000000000). ¡Guarda este token secreto!
2
🔌 Configurar credencial de Telegram en n8n
En n8n, ve a Credenciales → Nueva → Telegram API. Ponle un nombre (ej: "Mi Bot Telegram") y pega el Token que te dio BotFather. Guarda la credencial.
3
📨 Obtener tu Chat ID (para recibir mensajes)
Envía un mensaje a tu bot desde tu cuenta personal. Luego haz una petición a: https://api.telegram.org/bot<TU_TOKEN>/getUpdates. En la respuesta JSON, busca "chat":{"id": XXXXXXX}. Ese número es tu Chat ID.
4
⚙️ Configurar el nodo "Send a text message" en n8n (como en la imagen)
Dentro del workflow, añade nodo Telegram → "Send a text message". Selecciona la credencial que creaste, elige Resource: Message, Operation: Send Message. En el campo Chat ID coloca el número obtenido. En Text puedes usar expresiones como {{$json.mensaje_telegram}}. ¡Listo!
