Comandos de Docker

Guía rápida de referencia para la administración de n8n

🚀 Ciclo de Vida

# Levanta n8n y Postgres en segundo plano

docker-compose up -d

# Apaga los contenedores sin borrar tus datos

docker-compose down

# Apaga BORRANDO volúmenes (¡Resetea la BD!)

docker-compose down -v

# Reinicia ambos servicios rápidamente

docker-compose restart

🔍 Diagnóstico

# Muestra el estado actual de tus contenedores

docker-compose ps

# Ver qué está haciendo n8n en tiempo real

docker-compose logs -f n8n

# Ver los logs de la base de datos Postgres

docker-compose logs -f postgres

# Muestra el consumo de CPU y Memoria en vivo

docker stats

🧹 Limpieza y Acceso

# Elimina imágenes y contenedores sin usar (Libera espacio)

docker system prune -a

# Entrar a la consola interna del contenedor de n8n

docker-compose exec n8n ash

💾 Respaldos Rápidos

# Crea una copia de seguridad de la BD en un archivo .sql

docker-compose exec -t postgres pg_dumpall -U n8n_user > respaldo_n8n.sql

# Restaura el respaldo .sql (El contenedor debe estar activo)

cat respaldo_n8n.sql | docker-compose exec -T postgres psql -U n8n_user -d n8n_db
