✨ Flexbox — 1D Layout

🧩 Flexbox es ideal para interfaces unidimensionales (filas o columnas). Controla la alineación, distribución y orden de elementos hijos.
🔥 Prueba estas propiedades mágicas:
• justify-content: center; — centra horizontalmente
• flex-direction: column; — cambia a columna vertical
• align-items: stretch; — estira los items
• gap: 20px; — separación entre cajas
🧠 ¡Combínalas! 
.contenedor-flex {
  display: flex;
}

🧩 CSS Grid — 2D Layout

🏗️ Grid organiza elementos en filas Y columnas simultáneamente. Ideal para layouts complejos y estructuras de página.
🎯 Prueba estructuras Grid:
• grid-template-columns: repeat(3, 1fr); → 3 columnas iguales
• grid-template-rows: 80px 120px; → filas con altura
• gap: 15px; → separación general
• justify-items: center; → centra items horizontalmente 
.contenedor-grid {
  display: grid;
}

📖 Aprende a fondo: Flexbox vs CSS Grid

🔹 Flexbox (1D)

¿Qué es? Sistema unidimensional para distribuir espacio entre elementos de una interfaz. Opera sobre un eje principal (main axis) y un eje cruzado (cross axis).

  • flex-direction: Cambia la dirección (row | column | row-reverse).
  • justify-content: Alinea items en el eje principal (flex-start, center, space-between).
  • align-items: Alinea items en el eje cruzado (stretch, baseline, center).
  • flex-wrap: Permite que los elementos pasen a múltiples líneas.
  • Ideal para menús, barras laterales, tarjetas y componentes pequeños.

✨ Mentalidad: Piensa en Flexbox como una cinta elástica donde los elementos se reparten el espacio en una sola línea o varias.

🧱 CSS Grid (2D)

¿Qué es? Sistema bidimensional para diseñar layouts completos manejando filas y columnas al mismo tiempo. Basado en pistas (tracks).

  • grid-template-columns: Define tamaño y número de columnas (fr, %, px, auto, repeat()).
  • grid-template-rows: Define altura de filas.
  • gap: Espaciado entre celdas (anteriormente grid-gap).
  • justify-items / align-items: Alineación de items dentro de sus celdas.
  • Perfecto para estructurar páginas completas, galerías, dashboards y secciones complejas.

🎯 Diferencia clave: Grid trabaja en dos dimensiones simultáneamente (filas + columnas). Flexbox se especializa en una dimensión (fila O columna).

⚡ ¿Cuándo usar cada uno?

Usa Flexbox cuando necesites alinear elementos en una línea o distribuir espacio en componentes más pequeños (tarjetas, navegación, botones).

Usa CSS Grid cuando necesites control total sobre filas y columnas, por ejemplo: plantillas de página, calendarios, diseño de productos.

✅ Ambos se complementan: puedes tener un contenedor Grid y dentro usar Flexbox para componentes internos.

💡 Consejo: Para diseño responsive, combina Grid con @media queries y minmax() para adaptar columnas automáticamente.
