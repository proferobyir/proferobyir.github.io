Aprende Layouts Web Interactivos

Modifica el código CSS en los editores de la izquierda y observa instantáneamente el comportamiento de las cajas.

1. CSS Flexbox

Ideal para alinear elementos en una sola dimensión (filas o columnas). Prueba cambiando propiedades del contenedor.

Sugerencias para probar:
• justify-content: center; o space-between;
• flex-direction: column;
• align-items: center; 
.contenedor-flex {
  display: flex;
}

2. CSS Grid

Diseñado para estructuras bidimensionales complejas (filas y columnas simultáneamente).

Sugerencias para probar:
• grid-template-columns: repeat(3, 1fr);
• grid-template-columns: 1fr 2fr 1fr;
• gap: 20px; 
.contenedor-grid {
  display: grid;
}
