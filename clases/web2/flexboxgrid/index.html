Aprende a fondo: Flexbox vs CSS Grid

Flexbox (1D)

Que es? Sistema unidimensional para distribuir espacio entre elementos. Opera sobre eje principal y eje cruzado.

  • flex-direction: Cambia dirección (row | column).
  • justify-content: Alinea en eje principal (space-between, center).
  • align-items: Alinea en eje cruzado (stretch, center).
  • flex-wrap: Permite múltiples líneas.
  • Ideal para menús, barras laterales, componentes pequeños.

Mentalidad: una "cinta elástica" que organiza elementos en una dirección.

CSS Grid (2D)

Que es? Sistema bidimensional para manejar filas y columnas a la vez. Crea layouts complejos con pistas (tracks).

  • grid-template-columns: Define columnas (fr, %, auto, px).
  • grid-template-rows: Define filas.
  • gap: Espaciado entre celdas.
  • justify-items / align-items: Alineación en celdas.
  • Perfecto para páginas, dashboards, galerías.

Como definir el grid sin usar repeat(): En lugar de escribir repeat(3, 1fr), puedes listar manualmente cada valor. Por ejemplo: grid-template-columns: 1fr 1fr 1fr; genera tres columnas iguales. Si deseas una primera columna de 200px y el resto flexibles: grid-template-columns: 200px 1fr 2fr;. Sin la funcion repeat() obtienes el mismo control explícito y es perfecto para layouts personalizados o cuando las columnas no siguen un patrón repetitivo. Incluso puedes mezclar unidades: 150px auto minmax(100px, 1fr). La ventaja es la claridad visual de la estructura.

Diferencia clave: Grid = 2 dimensiones (filas+columnas), Flex = 1 dimensión (fila o columna).

Cuando usar cada uno?

Usa Flexbox para alinear elementos en línea (navegación, tarjetas, botones).

Usa CSS Grid para control total sobre filas y columnas (plantillas, calendarios, diseños complejos).

Se complementan: un contenedor Grid puede tener elementos Flex internos.

Consejo responsive sin repeat: combiná Grid con minmax() y media queries para adaptar columnas automáticamente. Por ejemplo en móvil defines grid-template-columns: 1fr; y en desktop cambias a 1fr 1fr 1fr.

Ejemplo sin repeat(): grid-template-columns: 250px 1fr 1fr; genera dos columnas de ancho flexible y una fija. Es totalmente valido y mas expresivo en algunos casos.

Definiendo CSS Grid sin la funcion repeat(): escritura manual avanzada

La funcion repeat() es comoda, pero no es obligatoria. Para definir una cuadricula sin ella, simplemente escribes los valores separados por espacios en grid-template-columns o grid-template-rows.

  • Dos columnas iguales: grid-template-columns: 1fr 1fr; (equivalente a repeat(2, 1fr)).
  • Tres columnas: primera fija 200px, segunda auto, tercera 1fr: grid-template-columns: 200px auto 1fr;
  • Columnas mixtas con medidas absolutas y relativas: grid-template-columns: 120px minmax(150px, 1fr) 2fr;
  • Definiendo filas especificas: grid-template-rows: 100px 200px auto; genera tres filas con alturas determinadas.

Este método es ideal cuando las dimensiones de las columnas no siguen un patron repetitivo o cuando deseas tener un control granular. Pruébalo en el editor de Grid: escribe por ejemplo grid-template-columns: 80px 1fr 2fr; y observa como la primera columna es angosta, la segunda ocupa el espacio restante proporcionalmente y la tercera el doble que la segunda. Sin repeat() ganas claridad para estructuras asimétricas.
