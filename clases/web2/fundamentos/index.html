Historia y evolución del desarrollo web

1989 - Web 1.0

Tim Berners-Lee: HTML, HTTP y URLs. Páginas estáticas, solo lectura. Primeros navegadores.

2004 - Web 2.0

JavaScript robusto, AJAX, contenido dinámico. Nacen las aplicaciones web interactivas.

2010+ - Web moderna

HTML5, CSS3 (Flexbox/Grid), frameworks, SPAs, WebAssembly, PWAs.

// Evolución: de etiquetas básicas a APIs complejas // 1995: JavaScript aparece para validación de formularios. // Hoy: control total sobre el DOM, canvas, WebRTC, etc.

Historia y fundamentos de JavaScript

1995 - Nacimiento

Brendan Eich crea Mocha → LiveScript → JavaScript. Dinamismo en Netscape.

ECMAScript (ES6+ )

let/const, arrow functions, clases, promesas, módulos. Estándar clave.

Características

Interpretado, tipado dinámico, basado en prototipos, asíncrono por naturaleza.

Tipos de datos, estructuras de control y funciones

Tipos primitivos: string, number, boolean, null, undefined, symbol, bigint.

Estructuras: if/else, switch, for, while, for...of, for...in.

Funciones: declarativas, expresivas, arrow functions, closures.

Programación orientada a objetos (POO) en la web

Clases y objetos

class MiClase { constructor() {} }
const obj = new MiClase();

Herencia / Polimorfismo

extends, super, métodos sobreescritos.

Encapsulación

Campos privados (#), getters/setters.

Ejemplo POO en acción: Clase `Usuario` y `Admin`.

// Fragmento de código POO class Usuario { constructor(nombre, email) { this.nombre = nombre; this.email = email; } saludar() { return `Hola, soy ${this.nombre}`; } } class Admin extends Usuario { constructor(nombre, email, nivel) { super(nombre, email); this.nivel = nivel; } saludarAdmin() { return `${this.saludar()} y tengo rol ${this.nivel}`; } }
