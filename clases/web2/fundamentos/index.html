1. Evolución del Ecosistema Web

De documentos planos e hipertextos estáticos a sistemas reactivos y distribuidos:

Web 1.0 (1989)

Tim Berners-Lee: Consumo estático, protocolos HTTP puros y documentos de sólo lectura.

Web 2.0 (2004)

Asincronía (AJAX), interactividad guiada por JS y el auge de las bases de datos dinámicas.

Web Moderna

HTML5 semántico, layouts dinámicos (Grid/Flexbox), SPAs, APIs nativas avanzadas y PWAs.
