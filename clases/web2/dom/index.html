📖 ¿Qué es el DOM?

El DOM (Document Object Model) es como un árbol genealógico que representa tu página HTML. Cuando el navegador carga una página, convierte cada etiqueta HTML en un objeto que JavaScript puede leer, modificar, agregar o eliminar.

🌳 Analogía: Imagina que tu HTML es una casa. El DOM es el plano de la casa: te dice dónde está cada habitación (etiqueta), qué hay dentro (contenido) y te permite remodelar sin construir desde cero.

Estructura del DOM

<!-- HTML -->
<html>
  <head><title>Mi página</title></head>
  <body>
    <h1>Hola</h1>
    <p>Texto</p>
  </body>
</html>

Se convierte en este árbol:

document
└── html
    ├── head
    │   └── title ── "Mi página"
    └── body
        ├── h1 ── "Hola"
        └── p  ── "Texto"
💡 Recuerda: Cada etiqueta es un nodo. JavaScript puede acceder a cualquier nodo usando el objeto document.

🎯 Selectores de elementos

Antes de modificar un elemento, necesitas seleccionarlo. JavaScript ofrece varias formas de hacerlo:

1. document.getElementById()

Selecciona un solo elemento por su atributo id. Es el más rápido y común. 

// HTML: <p id="saludo">Hola</p>
const elemento = document.getElementById("saludo");
console.log(elemento.textContent); // "Hola"

2. document.getElementsByClassName()

Selecciona todos los elementos que tengan una clase específica. Devuelve una colección (similar a un arreglo).

// HTML: <p class="rojo">...</p> <p class="rojo">...</p>
const rojos = document.getElementsByClassName("rojo");
console.log(rojos.length); // 2

3. document.getElementsByTagName()

Selecciona todas las etiquetas de un tipo (todos los p, todos los div, etc.). 

const parrafos = document.getElementsByTagName("p");

4. document.querySelector() ⭐

El más poderoso: usa selectores tipo CSS y devuelve el primer elemento que coincida.

const primero = document.querySelector(".clase");       // por clase
const unico   = document.querySelector("#miId");       // por id
const combo   = document.querySelector("div.rojo");    // div con clase rojo
const hijo    = document.querySelector("ul > li");     // li hijo directo de ul

5. document.querySelectorAll()

Igual que querySelector, pero devuelve todos los elementos que coincidan (como un arreglo).

const items = document.querySelectorAll("li.activo");
items.forEach(item => console.log(item.textContent));

📊 Comparativa rápida

Método Devuelve Selector
getElementById 1 elemento id
getElementsByClassName Colección clase
getElementsByTagName Colección etiqueta
querySelector 1 elemento CSS (cualquiera)
querySelectorAll Colección CSS (cualquiera)

✍️ innerHTML vs textContent

Ambas propiedades permiten leer o cambiar el contenido de un elemento, pero se comportan distinto:

textContent

Trata todo como texto plano. Si pones etiquetas HTML, se mostrarán como texto literal. 

elemento.textContent = "<b>Hola</b>";
// Resultado visible: <b>Hola</b>

innerHTML

Interpreta el HTML. Puedes insertar etiquetas y se renderizarán.

elemento.innerHTML = "<b>Hola</b>";
// Resultado visible: Hola (en negrita)
⚠️ Cuidado con innerHTML: Si insertas contenido que viene del usuario, un atacante podría inyectar código malicioso (XSS). Usa textContent cuando no necesites HTML.

🧱 Agregar y eliminar nodos hijos

El DOM es dinámico: puedes crear, agregar y eliminar elementos en tiempo real.

Crear un elemento

const nuevo = document.createElement("p");
nuevo.textContent = "Soy nuevo";

Agregar como hijo

  • padre.appendChild(nuevo) → lo agrega al final.
  • padre.prepend(nuevo) → lo agrega al inicio.
  • padre.insertBefore(nuevo, referencia) → antes de un elemento específico.
  • padre.append("texto", nuevo) → acepta varios elementos o texto.

Eliminar un elemento

// Forma moderna (la más fácil):
elemento.remove();

// Forma clásica (desde el padre):
padre.removeChild(hijo);

📦 Declaración de variables en JavaScript

Una variable es una caja con nombre donde guardas información.

Las 3 formas de declarar

Palabra ¿Se puede reasignar? ¿Se puede redeclarar? Ámbito
var ✅ Sí ✅ Sí Función (antiguo, evita usarlo)
let ✅ Sí ❌ No Bloque {}
const ❌ No ❌ No Bloque {} 
var nombre = "Ana";      // antiguo, mejor evitar
let edad = 25;            // puede cambiar
const PI = 3.1416;        // constante, no cambia

edad = 26;                // ✅ OK
// PI = 3;                // ❌ Error
💡 Regla de oro: Usa const por defecto. Solo usa let cuando sepas que el valor va a cambiar. Evita var.

📚 Arreglos (Arrays)

Un arreglo es una lista ordenada de valores. Se declara con corchetes []. 

const frutas = ["manzana", "pera", "uva"];
const numeros = [10, 20, 30, 40];
const mixto = ["hola", 42, true];

Acceder por índice (empieza en 0)

console.log(frutas[0]); // "manzana"
console.log(frutas[2]); // "uva"
console.log(frutas.length); // 3

Métodos útiles

frutas.push("kiwi");       // agrega al final
frutas.pop();              // elimina el último
frutas.unshift("mango");   // agrega al inicio
frutas.shift();            // elimina el primero
frutas.includes("pera");  // true

🗂️ Arreglos JSON (arrays de objetos)

En la práctica, casi siempre trabajamos con arreglos de objetos. Cada objeto representa un registro con varias propiedades.

const estudiantes = [
  { nombre: "Ana",   edad: 20, carrera: "Sistemas" },
  { nombre: "Luis",  edad: 22, carrera: "Diseño" },
  { nombre: "María", edad: 19, carrera: "Medicina" }
];

Acceder a datos

console.log(estudiantes[0].nombre);   // "Ana"
console.log(estudiantes[1].carrera);  // "Diseño"
💡 JSON significa JavaScript Object Notation. Es el formato estándar para intercambiar datos en la web.

🔁 Bucles

Los bucles repiten código varias veces. Son esenciales para recorrer arreglos.

1. for clásico

for (let i = 0; i < 5; i++) {
  console.log(i); // 0, 1, 2, 3, 4
}

2. while

let n = 3;
while (n > 0) {
  console.log(n);
  n--;
}

3. for...of (ideal para arreglos)

const colores = ["rojo", "verde", "azul"];
for (const color of colores) {
  console.log(color);
}

4. forEach (método de arreglos)

colores.forEach((color, indice) => {
  console.log(indice, color);
});

🚀 Recorrer arreglos JSON con registros estáticos

Combinemos todo: un arreglo JSON + un bucle + manipulación del DOM.

Ejemplo completo: mostrar estudiantes en una tabla

const estudiantes = [
  { nombre: "Ana",   edad: 20, carrera: "Sistemas" },
  { nombre: "Luis",  edad: 22, carrera: "Diseño" },
  { nombre: "María", edad: 19, carrera: "Medicina" }
];

const contenedor = document.querySelector("#tabla");

estudiantes.forEach(est => {
  const fila = document.createElement("tr");
  fila.innerHTML = `
    <td>${est.nombre}</td>
    <td>${est.edad}</td>
    <td>${est.carrera}</td>
  `;
  contenedor.appendChild(fila);
});

Ejemplo 2: tarjetas dinámicas

const productos = [
  { nombre: "Laptop",  precio: 1500 },
  { nombre: "Mouse",   precio: 25 },
  { nombre: "Teclado", precio: 80 }
