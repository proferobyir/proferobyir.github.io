JavaScript: El lenguaje de la web

¿Qué es JavaScript? Es un lenguaje de programación que permite agregar interactividad, dinamismo y comportamiento a las páginas web. Mientras que HTML da estructura y CSS da estilo, JavaScript controla el comportamiento: responder a clics, modificar contenido, comunicarse con servidores, animar elementos, etc.

¿Cómo incluir JavaScript en un archivo HTML?

Existen tres formas principales de incluir código JavaScript en un documento HTML:

1. JavaScript interno (dentro de la etiqueta script)

<!DOCTYPE html>
<html>
<head>
    <script>
        console.log("Hola desde JavaScript interno");
    </script>
</head>
<body>...</body>
</html>

2. JavaScript externo (recomendado para proyectos grandes)

<script src="mi-script.js"></script>

El archivo mi-script.js contendrá todo el código JavaScript. Esto mantiene HTML limpio y permite reutilizar el código.

3. JavaScript inline (en atributos de eventos, no recomendado)

<button onclick="alert('Clic')">Click</button>
📚 IMPORTANTE: El orden de carga y el atributo 'defer'

Cuando el navegador lee HTML y encuentra una etiqueta <script>, detiene el procesamiento del HTML para descargar y ejecutar el JavaScript. Esto puede causar problemas si el script intenta acceder a elementos del DOM que aún no han sido cargados.

¿Qué es 'defer'? Es un atributo que se coloca en la etiqueta <script> para indicar al navegador que:

  • Descargue el script en paralelo mientras sigue parseando el HTML
  • Ejecute el script SOLO después de que el HTML esté completamente cargado
<script src="mi-script.js" defer></script>

¿Qué es el evento 'DOMContentLoaded'? Es un evento que se dispara cuando el navegador ha terminado de cargar y parsear completamente el HTML (sin esperar hojas de estilo, imágenes u otros recursos).

document.addEventListener('DOMContentLoaded', function() {
    // Este código se ejecuta SEGURO cuando el DOM está listo
});

Selectores en CSS/HTML: Clases vs IDs

Antes de aprender a seleccionar elementos con JavaScript, es fundamental entender cómo identificamos los elementos en HTML usando class y id.

🆔 ¿Qué es un ID (#id)?

DEFINICIÓN: Un id (identificador) es un atributo que se asigna a un elemento HTML para darle un nombre ÚNICO dentro de toda la página. NO puede repetirse.

Sintaxis en HTML:

<div id="miContenedor">...</div>
<button id="btnEnviar">Enviar</button>
<p id="parrafoPrincipal">Texto importante</p>

Cómo se ve en CSS (para estilos): Se selecciona con # + nombre del id

/* CSS */
#miContenedor {
    background-color: blue;
}
#btnEnviar {
    color: white;
}

Cómo se selecciona en JavaScript:

// JavaScript
const elemento = document.getElementById('miContenedor');
// o también:
const elemento2 = document.querySelector('#miContenedor');
⚠️ REGLA DE ORO: Un mismo id NO puede usarse en más de un elemento. ¡Es como tu número de identificación personal!

🔘 ¿Qué es una clase (.clase)?

DEFINICIÓN: Una class (clase) es un atributo que se asigna a elementos HTML para agruparlos bajo un mismo nombre. SÍ puede repetirse en múltiples elementos.

Sintaxis en HTML:

<div class="tarjeta">Tarjeta 1</div>
<div class="tarjeta">Tarjeta 2</div>
<div class="tarjeta destacada">Tarjeta especial</div>
<p class="texto-error">Mensaje de error</p>

Cómo se ve en CSS (para estilos): Se selecciona con . + nombre de la clase 

/* CSS */
.tarjeta {
    border: 1px solid gray;
    padding: 10px;
}
.destacada {
    background-color: yellow;
}
.texto-error {
    color: red;
}

Cómo se selecciona en JavaScript:

// JavaScript - selecciona el PRIMER elemento con esa clase
const primeraTarjeta = document.querySelector('.tarjeta');

// JavaScript - selecciona TODOS los elementos con esa clase
const todasLasTarjetas = document.querySelectorAll('.tarjeta');

📊 Comparativa Rápida: ID vs Clase

Característica ID (#) Clase (.)
Unicidad ÚNICO por página Múltiples elementos
Sintaxis selector #miId .miClase
Método JS rápido getElementById() getElementsByClassName()
Selector versátil JS querySelector('#miId') querySelector('.miClase') o querySelectorAll('.miClase')
Uso típico Elementos únicos (header, footer, formulario principal) Grupos de elementos (tarjetas, botones, mensajes)

El DOM (Document Object Model)

¿Qué es el DOM? Es una representación en árbol de tu documento HTML. Cada etiqueta se convierte en un nodo que JavaScript puede leer, modificar, crear o eliminar.

SELECTORES DEL DOM: Explicación con ejemplos de este código

🔍 1. document.getElementById('id')

Selecciona un ÚNICO elemento por su id. Es el método más rápido. Retorna el elemento o null si no existe. 

// En este playground:
const container = document.getElementById('dynamicNodesContainer');
// Busca el elemento con id="dynamicNodesContainer" (el contenedor de tarjetas)
🔍 2. document.querySelector('.clase')

Selecciona el PRIMER elemento que coincida con un selector CSS (puede ser clase, id, etiqueta, etc.) 

// En este playground:
document.querySelector('.dynamic-card')
// Busca la PRIMERA tarjeta con clase 'dynamic-card'
🔍 3. document.querySelectorAll('.clase')

Selecciona TODOS los elementos que coinciden con el selector CSS. Retorna una NodeList. 

// En este playground:
const nodosHijos = container.querySelectorAll('.dynamic-card');
// Selecciona TODAS las tarjetas dentro del contenedor

Operaciones básicas: crear y borrar nodos

// Crear un nuevo elemento
const nuevoDiv = document.createElement('div');
nuevoDiv.textContent = 'Hola Mundo';

// Añadirlo como hijo
document.body.appendChild(nuevoDiv);

// Eliminar un nodo
const elementoAEliminar = document.querySelector('#viejo');
elementoAEliminar.remove();

Variables en JavaScript, console.log y alcance (scope)

Declaración de variables

let → variable que puede cambiar (alcance de bloque).
const → constante, no se reasigna.
var → forma antigua, evitar.

console.log

console.log("Mensaje", miVariable);
Muestra valores en la consola del navegador (F12).

Alcance (scope)

Global: visible en todo el script.
Local: dentro de una función o bloque. 

// Ejemplo de scope
let global = "Soy global";

function miFuncion() {
    let local = "Solo dentro de la función";
    console.log(global); // Sí accede
}
// console.log(local); // Error: local no está definida fuera
