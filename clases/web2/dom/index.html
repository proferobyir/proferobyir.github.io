JavaScript: El lenguaje de la web

¿Qué es JavaScript? Es un lenguaje de programación que permite agregar interactividad, dinamismo y comportamiento a las páginas web. Mientras que HTML da estructura y CSS da estilo, JavaScript controla el comportamiento: responder a clics, modificar contenido, comunicarse con servidores, animar elementos, etc.

¿Cómo incluir JavaScript en un archivo HTML?

Existen tres formas principales de incluir código JavaScript en un documento HTML:

1. JavaScript interno (dentro de la etiqueta script)

<!DOCTYPE html>
<html>
<head>
    <script>
        console.log("Hola desde JavaScript interno");
    </script>
</head>
<body>...</body>
</html>

2. JavaScript externo (recomendado para proyectos grandes)

<script src="mi-script.js"></script>

El archivo mi-script.js contendrá todo el código JavaScript. Esto mantiene HTML limpio y permite reutilizar el código.

3. JavaScript inline (en atributos de eventos, no recomendado)

<button onclick="alert('Clic')">Click</button>

Importante: El orden de carga importa. Si el script manipula el DOM, se recomienda colocarlo antes del cierre de </body> o usar defer/DOMContentLoaded.

El DOM (Document Object Model)

¿Qué es el DOM? Es una representación en árbol de tu documento HTML. Cada etiqueta se convierte en un nodo que JavaScript puede leer, modificar, crear o eliminar. El DOM es la interfaz que permite que JavaScript interactúe con la página.

Conceptos clave del DOM

  • Nodo raíz: document representa toda la página.
  • Nodos elemento: cada etiqueta HTML (div, p, button...).
  • Nodos texto: el contenido textual dentro de las etiquetas.

Selectores del DOM

Para acceder a un elemento necesitamos selectores. Métodos principales: 

// Por ID
const titulo = document.getElementById('miTitulo');

// Por selector CSS (el más versátil)
const contenedor = document.querySelector('.contenedor');
const todosLosItems = document.querySelectorAll('.item');

// Por clase
const elementos = document.getElementsByClassName('clase');

Operaciones básicas: crear y borrar nodos

// Crear un nuevo elemento
const nuevoDiv = document.createElement('div');
nuevoDiv.textContent = 'Hola Mundo';

// Añadirlo como hijo
document.body.appendChild(nuevoDiv);

// Eliminar un nodo
const elementoAEliminar = document.querySelector('#viejo');
elementoAEliminar.remove();   // método moderno
// o: padre.removeChild(hijo);

Variables en JavaScript, console.log y alcance (scope)

Declaración de variables

let → variable que puede cambiar (alcance de bloque).
const → constante, no se reasigna.
var → forma antigua, evitar (alcance de función).

console.log

console.log("Mensaje", miVariable);
Muestra valores en la consola del navegador (F12 → pestaña Consola). Esencial para depurar.

Alcance (scope)

Global: visible en todo el script.
Local: dentro de una función o bloque { } con let/const. 

// Ejemplo de scope
let global = "Soy global";

function miFuncion() {
    let local = "Solo dentro de la función";
    console.log(global); // Sí accede
    console.log(local);  // Sí accede
}
// console.log(local); // Error: local no está definida fuera
