¿Qué es el DOM (Document Object Model)?

Que es el DOM? Es una representacion en arbol de tu documento HTML. Cada etiqueta es un nodo que JS puede modificar, crear o eliminar. El DOM permite que las paginas web sean dinamicas e interactivas.

Fundamentos JS + DOM

Variables y scope

let / const / var
let nombre = "Ana"; (alcance de bloque)
const PI = 3.14; (no reasignable)
El scope global vs local (funcion/bloque).

console.log

Muestra valores en consola del navegador (F12). Ejemplo:
console.log("valor:", miVariable); -> Util para depurar.

Selectores del DOM

document.getElementById(), querySelector(), querySelectorAll().
Ej: const caja = document.querySelector("#miCaja");

Manipulacion de nodos

createElement(), appendChild(), removeChild(), remove(), innerHTML.
