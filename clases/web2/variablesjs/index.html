🔌 1. Inclusión de Scripts: Defer vs Async

Controla cómo se descargan los archivos externos de JavaScript sin congelar la carga de la página:

<!-- Se ejecuta en segundo plano y espera a que el HTML esté listo -->
<script src="script.js" defer></script>

<!-- Se ejecuta inmediatamente en cuanto termina de descargarse -->
<script src="script.js" async></script>

📦 2. Variables y Tipos de Datos

Usamos let para valores modificables y const para valores fijos protegidos.

let nombre = "Carlos";
const edad = 28;

🔍 3. Scope de Bloque (Visualización por Bordes)

Mira cómo los bordes de color delimitan de forma perfecta el ciclo de vida de cada variable. Lo que nace dentro del borde rosa, muere al cerrarse ese borde:

let global = "Vivo en todo el archivo"; // ← Nace la global

if (true) {
let local = "Solo vivo dentro de estas llaves"; // ← Nace la local
console.log(local); // ✅ Funciona perfectamente aquí dentro
} // ← Aquí se cierra el bloque local
console.log(global); // ✅ Funciona (seguimos dentro del borde azul)
console.log(local); // ❌ ¡ERROR! Fuera del borde rosa, la variable 'local' ya no existe.

🛠️ 6. Creación de Funciones Tradicionales y Flecha

Compara la estructura clásica con la palabra reservada function frente a la sintaxis limpia de flecha:

Sintaxis Tradicional:

function calcularDoble(numero) {
    return numero * 2;
}

Sintaxis Flecha (Arrow Function):

const calcularDobleFlecha = (numero) => {
    return numero * 2;
};
