🔌 1. Inclusión de Scripts: Defer vs Async

Cuando vinculamos un archivo externo en el <head>, usamos atributos para controlar la carga:

defer: Descarga el script de fondo y lo ejecuta solo cuando el HTML está 100% cargado. Mantiene el orden de los scripts.

async: Descarga el script y lo ejecuta de inmediato, interrumpiendo la carga del HTML.

📦 2. Variables (let, const) y Tipos de Datos

Guardamos información en contenedores. let permite cambiar el valor más adelante; const bloquea el valor para siempre.

let nombreEstudiante = "Tú"; 
const edadMinima = 18;

🔍 3. Scope de Bloque (Vectores de Alcance)

El "Scope" determina la zona donde una variable es accesible. Observa el código y el mapa vectorial inferior para entender dónde nace y dónde muere cada una:

let global = "Visible en todos lados";

if (true) {
    let local = "Solo vivo dentro de las llaves";
    console.log(local);  // ✅ Funciona
} 
// ❌ AQUÍ MUERE LA VARIABLE 'local'

console.log(global); // ✅ Funciona
// console.log(local); // ❌ ¡ERROR! Fuera de su bloque ya no existe.

🔀 4. Operadores Lógicos y Condicionales

Usamos el operador && (AND) para verificar que se cumplan dos condiciones al mismo tiempo, y sentencias if / else para tomar caminos diferentes.

🔁 5. Ciclos (Loops)

El bucle for nos permite repetir una acción de forma automática usando un contador incremental.

🏹 6. Funciones Tradicionales vs Funciones Flecha

Las funciones agrupan líneas de código reutilizables. Compara sus estructuras:

// Tradicional
function duplicar(n) { return n * 2; }

// Función Flecha (Moderna)
const duplicarFlecha = (n) => n * 2;
