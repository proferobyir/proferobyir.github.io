📦 Declaración y Tipos de Variables

Una variable es un contenedor para almacenar datos. En JavaScript moderno tenemos tres formas de crearlas, aunque sólo dos son recomendadas.

💡 Analogía: Imagina que una variable es una caja etiquetada. Con let puedes sacar lo que tiene dentro y meter algo nuevo. Con const, la caja se sella con candado una vez que guardas algo en ella.

Las 3 formas de declarar

Palabra clave ¿Reasignable? ¿Redeclarable? Comportamiento
var ✅ Sí ✅ Sí Antigua, causa problemas de scope. ¡Evítala!
let ✅ Sí ❌ No Ideal para valores que sabes que van a cambiar.
const ❌ No ❌ No Constante. Úsala por defecto para código seguro.

Tipos de Datos Primitivos Comunes

let nombre = "Carlos";     // String (Texto)
const edad = 28;            // Number (Número)
let esMayor = true;         // Boolean (Verdadero/Falso)
let vacio = null;           // Null (Intencionalmente vacío)
let indefinido;             // Undefined (Declarada pero sin valor)

🔍 Ámbito o Alcance (Scope)

El Scope determina la accesibilidad (visibilidad) de las variables. Las variables declaradas con let y const tienen lo que llamamos Scope de Bloque.

⚠️ Scope de Bloque {}: Cualquier variable declarada dentro de llaves (como en un if o un bucle) no puede ser accedida desde fuera de esas llaves. 
let global = "Estoy afuera";

if (true) {
    let local = "Estoy adentro";
    console.log(global); // ✅ Funciona
}
// console.log(local); // ❌ Error: local no está definida aquí

🚦 Sentencias Condicionales (if, else if, else)

Las condicionales permiten que tu código tome caminos diferentes basados en si una condición se cumple o no.

let hora = 14;

if (hora < 12) {
    console.log("Buenos días");
} else if (hora < 19) {
    console.log("Buenas tardes");
} else {
    console.log("Buenas noches");
}

🔁 Ciclos (Loops)

Los ciclos nos permiten repetir un bloque de código múltiples veces sin necesidad de escribir lo mismo una y otra vez.

1. El ciclo for

Ideal cuando sabes exactamente cuántas veces quieres repetir la tarea.

for (let i = 1; i <= 3; i++) {
    console.log("Vuelta número " + i);
}

2. El ciclo while

Se ejecuta mientras una condición específica sea verdadera. ¡Cuidado con los bucles infinitos!

let energia = 3;
while (energia > 0) {
    console.log("Cargando...");
    energia--;
}
