🔌 Cómo agregar JavaScript a HTML

Para incluir código externo en el <head> disponemos de dos atributos clave que evitan que la página se congele al cargar:

📦 Defer: Descarga el script en segundo plano y lo ejecuta al final, cuando todo el HTML está listo. Mantiene el orden de tus archivos.
🚀 Async: Descarga el script y lo ejecuta inmediatamente en cuanto termina de bajarse, interrumpiendo el HTML si hace falta.

📦 Declaración y Tipos de Variables

Usamos let para cajas cuyo contenido cambia y const para cajas cerradas con candado que nunca cambian.

let nombre = "Carlos"; // String
const edad = 28;      // Number
let esMayor = true;   // Boolean

🔍 Ámbito o Alcance (Scope)

Las variables declaradas con let dentro de llaves {} mueren ahí dentro; no existen afuera.

let global = "Estoy afuera";
if (true) {
    let local = "Estoy adentro";
}
// console.log(local); // ❌ ¡Dará un error!

🔀 Operadores Lógicos

El operador && (AND) obliga a que obligatoriamente se cumplan los dos lados para dar un resultado verdadero.

let edadPersona = 19;
let tieneEntrada = false; // (Cambia este valor con el botón)
let acceso = (edadPersona >= 18) && tieneEntrada;

🚦 Sentencias Condicionales (if / else if)

Permiten que el flujo del programa tome decisiones según la variable que analice.

let semaforo = "Rojo"; // (Elige el color con los botones)
if (semaforo === "Rojo") {
    resultado = "🛑 ¡Frena!";
} else if (semaforo === "Verde") {
    resultado = "🍏 ¡Pasa!";
}

🔁 Ciclos (Loops)

El ciclo for repite instrucciones sumando de uno en uno usando un contador (i++).

for (let i = 1; i <= 3; i++) {
    console.log("Vuelta número " + i);
}
