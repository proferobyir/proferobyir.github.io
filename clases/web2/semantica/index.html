🎨 Laboratorio CSS: Clases, IDs y Especificidad

📌 ¿Qué es una CLASE en CSS?

Una clase es un selector que puedes reutilizar en múltiples elementos. Se define con un punto .nombre-clase en CSS y se aplica con el atributo class="nombre-clase" en HTML.

.texto-verde { color: green; } ← Esto es una CLASE

✅ Ventaja: Puedes usarla en varios elementos a la vez. ¡Es reutilizable!

📌 ¿Qué es un ID en CSS?

Un ID es un selector único que solo puede usarse UNA VEZ por página. Se define con #nombre-id en CSS y se aplica con id="nombre-id".

#texto-especial-id { text-transform: uppercase; } ← Esto es un ID (único)

⚠️ Importante: Un ID es más específico que una clase. Si hay conflicto, gana el ID.

⚡ Especificidad (¿quién gana?)

CSS tiene un sistema de pesos: ID (100 puntos) > CLASE (10 puntos) > etiqueta (1 punto). El selector más específico es el que se aplica.
