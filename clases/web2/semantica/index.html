📖 Conceptos fundamentales

Definición

SEO

Search Engine Optimization: conjunto de técnicas para posicionar una web en los resultados de Google.

🎯 Las etiquetas semánticas ayudan al SEO porque los buscadores entienden mejor tu contenido.
Definición

Crawler

Un crawler (araña web) es un robot que recorren internet analizando el código de las páginas.

🕷️ Googlebot lee tus etiquetas semánticas para indexar tu sitio correctamente.

🏷️ Etiquetas semánticas HTML5

<header>

header

Cabecera del sitio. Contiene la marca, navegación y elementos introductorios.

📈 Ayuda al crawler a identificar la zona de marca y el contenido principal de la cabecera.
<nav>

nav

Menú de navegación principal o secundaria del sitio.

🔗 Los crawlers descubren enlaces importantes y la estructura de navegación.
<main>

main

Contenido principal y único de la página. Solo debe haber uno por documento.

⭐ Google prioriza el texto dentro de main como el contenido más relevante.
<article>

article

Contenido independiente y autónomo como noticias, posts o entradas de blog.

📰 Ideal para contenido que puede distribuirse o sindicarse de forma independiente.
<section>

section

Agrupa contenido temático relacionado dentro de un documento.

📚 Organiza el documento en capítulos lógicos que los crawlers pueden interpretar.
<aside> <footer>

aside / footer

Contenido complementario (aside) y pie de página (footer).

🧩 Clarifica contenido secundario y proporciona contexto adicional a los buscadores.

🎨 Laboratorio CSS: Clases, IDs y Especificidad

🧪 ¡Experimenta! Haz clic en los botones y mira cómo cambia el texto

🦄 ¡Hola, estudiantes! Este texto cambiará de estilo con CSS.

🔍 ¿Qué está pasando?

▶️ El botón "Aplicar clase .texto-verde" añade la clase texto-verde al div. Esto cambia el color a verde.
▶️ El botón "Aplicar ID" añade el ID texto-especial-id. Como los IDs tienen más especificidad, su estilo puede ganar sobre las clases.
▶️ El botón Resetear elimina TODAS las clases e IDs, volviendo al estado original.
▶️ Concepto clave: Si aplicas el ID y luego una clase, el ID puede prevalecer porque es más específico.

💡 Pista para estudiantes: Abre las herramientas de desarrollador (F12) y observa cómo cambian los atributos class e id en el HTML.

📚 Resumen de la clase:

  • SEO: Posicionamiento en buscadores. Crawler: Robot que lee tu web.
  • Etiquetas semánticas: header, nav, main, article, section, aside, footer → mejoran el SEO.
  • Clase CSS (.): Reutilizable, se aplica con class="nombre".
  • ID CSS (#): Único por página, más específico, se aplica con id="nombre".
  • Especificidad: ID (100) > Clase (10) > Etiqueta (1).
