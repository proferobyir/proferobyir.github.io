Paso 1 El Concepto de Asincronía

En JavaScript, operaciones como consultar un servidor web tardan tiempo en completarse. Para evitar que la interfaz se bloquee, utilizamos funciones asíncronas basadas en Promesas.

Paso 2 La estructura Async / Await

La combinación async/await nos permite procesar la espera del servidor de forma ordenada, capturando las respuestas dentro de bloques de control de errores try...catch.

Paso 3 Petición de Datos (Fetch)

Estructura lógica utilizada para conectarse al servidor y retornar el arreglo crudo de personajes extraídos de la API externa:

async function obtenerPersonajes() {
    try {
        const respuesta = await fetch('https://dragonball-api.com/api/characters');
        const datos = await respuesta.json();
        return datos.items; // Retorna el arreglo
    } catch (error) {
        console.error(error);
    }
}

Paso 4 Renderizado: Pintando los Datos en el Frontend

Una vez que la promesa se resuelve y tenemos los datos en memoria, el siguiente paso crítico en el desarrollo web es la manipulación del DOM. Necesitamos transformar objetos de JavaScript en elementos visuales HTML reales.

1. El contenedor base (HTML): En nuestra vista preparamos un elemento vacío con un identificador único (id), que servirá como el anclaje donde se "inyectarán" dinámicamente las tarjetas:

<div class="cards-container" id="contenedorCards">
    <!-- Las tarjetas de los personajes se sembrarán aquí mediante código -->
</div>

2. La función de Renderizado (JavaScript): Esta función recibe el arreglo de datos, limpia los mensajes de carga previos utilizando innerHTML = '', y recorre uno a uno los elementos para maquetar la interfaz usando Template Literals (comillas invertidas `):

function renderizarPersonajes(personajes) {
    // 1. Seleccionamos el contenedor HTML de destino
    const destino = document.getElementById('contenedorCards');
    destino.innerHTML = ''; // Limpiamos el contenedor

    // 2. Iteramos sobre el arreglo de datos procesado de la API
    personajes.forEach(personaje => {
        
        // 3. Creamos un nuevo elemento contenedor 'div' en memoria
        const card = document.createElement('div');
        card.classList.add('character-card'); // Asignamos clase Flexbox

        // 4. Estructuramos el contenido HTML interno mapeando las propiedades del objeto
        card.innerHTML = `
            <img src="${personaje.image}" alt="${personaje.name}">
            <div class="character-info">
                <h4>${personaje.name}</h4>
                <p>Ki: ${personaje.ki}</p>
                <p>Raza: ${personaje.race}</p>
            </div>
        `;
        
        // 5. Insertamos la tarjeta física dentro del nodo padre del DOM
        destino.appendChild(card);
    });
}
