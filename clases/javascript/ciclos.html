📊 Representación Gráfica de un Bucle

💡 Explicación: El bucle comienza inicializando una variable (i=0). Luego verifica la condición: si es verdadera, ejecuta el código, incrementa la variable y vuelve a evaluar. Cuando la condición es falsa, sale del bucle.

📈 Evolución de la variable en un bucle

🔑 Concepto clave: La variable de control (i) cambia en cada iteración. Inicia en 0, luego 1, 2, 3... hasta que la condición se vuelve falsa.

🎯 Ejemplo 1: Bucle FOR + Evento onclick

Ingresa un número y al hacer clic, el bucle mostrará todos los números desde 1 hasta ese límite.

Esperando acción...
document.getElementById("btnMostrarNumeros").onclick = function() {
    let limite = document.getElementById("limiteNumero").value;
    let salida = "";
    for (let i = 1; i <= limite; i++) {
        salida += i + " ";
    }
    document.getElementById("resultadoFor").innerHTML = "🔢 " + salida;
};

🎯 Ejemplo 2: Bucle WHILE + Evento oninput

Escribe un número y en tiempo real verás su tabla de multiplicar usando un bucle while.

La tabla aparecerá aquí...
document.getElementById("numeroTabla").oninput = function() {
    let num = parseInt(this.value) || 0;
    let i = 1;
    let tabla = "";
    while(i <= 10) {
        tabla += `${num} x ${i} = ${num * i}\n`;
        i++;
    }
    document.getElementById("resultadoWhile").innerHTML = tabla;
};

⚠️ Errores comunes con ciclos

🔥 Bucle infinito
for(let i=0; i>=0; i++){ }

❌ La condición siempre es verdadera.

📌 Olvidar incremento
let i=0; while(i<5){ }

❌ i nunca cambia → bucle infinito.

🎯 Confundir = con ==
if(i = 5)

✅ Usar === para comparar.

💡 Variable no declarada

Siempre declarar la variable del bucle con let.
