🔄 BUCLE FOR

📌 FOR: inicialización → condición → {código} → incremento → repite

🔄 BUCLE WHILE

📌 WHILE: Evalúa condición ANTES de ejecutar. Si es falsa inicial, no entra.

🔄 BUCLE DO...WHILE

📌 DO...WHILE: ejecuta al menos una vez, luego evalúa condición.

📦 Estructuras de Datos: Arrays y su Recorrido

Un Array (arreglo/vector) contiene múltiples valores indexados consecutivamente empezando desde la posición 0. Para leer cada recuadro, usamos un ciclo for donde el índice avanza dinámicamente.

const tecnologias = ["HTML", "CSS", "JS", "n8n"];
for (let i = 0; i < tecnologias.length; i++) {
    console.log("Elemento en posición " + i + ": " + tecnologias[i]);
}
