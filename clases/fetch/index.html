1. ¿Qué es JSON?

JSON (JavaScript Object Notation) es un formato ligero para intercambiar datos.

Objeto JSON

{ "id":1, "nombre":"Juan", "edad":25 }

Arreglo JSON

[ { "id":1, "nombre":"Juan" }, { "id":2, "nombre":"Ana" } ]
