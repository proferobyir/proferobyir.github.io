Aprendiendo Async/Await, Promise.all y métodos de array como .map() y .forEach() para transformar arreglos JSON

📚 ¿Qué es el método .map()?

.map() es un método de los arreglos en JavaScript que crea un NUEVO arreglo aplicando una función a cada elemento del arreglo original. Es perfecto para transformar datos JSON.

// ARREGLO JSON ORIGINAL (Pokémon) const pokemones = [ { nombre: "Pikachu", tipo: "eléctrico", nivel: 25 }, { nombre: "Charmander", tipo: "fuego", nivel: 16 } ]; // .map() transforma CADA elemento y devuelve un NUEVO arreglo const soloNombres = pokemones.map(pokemon => { return pokemon.nombre; // 👈 Extrae solo el nombre }); // Resultado: ["Pikachu", "Charmander"] // También podemos transformar la estructura: const tarjetasHTML = pokemones.map(p => `
${p.nombre} - ${p.tipo}
` );
💡 Dato clave: map() NO modifica el arreglo original, crea uno nuevo. Ideal para transformar datos de APIs.

🌐 2. .map() + PokéAPI (transformación)

Usamos .map() sobre un arreglo de nombres para crear múltiples promesas de fetch.

// Arreglo con nombres de Pokémon const nombresPokemon = ["pikachu", "charizard", "mewtwo", "eevee"]; // 🔥 .map() convierte CADA nombre en una Promesa de fetch async function cargarVarios() { // map() crea un NUEVO arreglo con las promesas const promesas = nombresPokemon.map(nombre => { return fetch(`https://pokeapi.co/api/v2/pokemon/${nombre}`) .then(res => res.json()); }); // Promise.all ejecuta todas las promesas en paralelo const resultados = await Promise.all(promesas); // OTRO .map() para extraer SOLO lo que necesitamos const datosLimpios = resultados.map(pokemon => ({ nombre: pokemon.name, imagen: pokemon.sprites.front_default, tipo: pokemon.types[0].type.name })); return datosLimpios; }
🌟 Usamos .map() para transformar nombres → Promesas → Datos con imágenes.

🔄 3. .map() vs .forEach()

.map() retorna un NUEVO arreglo. .forEach() solo itera, no retorna nada.

// DIFERENCIA PRÁCTICA: // ❌ forEach() NO retorna un nuevo arreglo const numeros = [1, 2, 3, 4, 5]; const resultadoForEach = numeros.forEach(n => n * 2); console.log(resultadoForEach); // undefined ❌ // ✅ map() SÍ retorna un nuevo arreglo const resultadoMap = numeros.map(n => n * 2); console.log(resultadoMap); // [2, 4, 6, 8, 10] ✅ // En PokéAPI: .map() es ideal para transformar datos const nombres = ["pikachu", "bulbasaur"]; const mayusculas = nombres.map(n => n.toUpperCase()); // Resultado: ["PIKACHU", "BULBASAUR"]
💡 Haz clic para ver la diferencia en la consola del navegador (F12).

⛓️ 4. Cadena de .map() (transformaciones múltiples)

Podemos encadenar varios .map() para hacer transformaciones paso a paso.

🔍 5. map() + filter() (búsqueda inteligente)

Combinamos .filter() (filtra elementos) y .map() (transforma resultados).

// Filtrar Pokémon por tipo y luego mapear sus nombres async function buscarPorTipo(tipoBuscado) { const respuesta = await fetch('https://pokeapi.co/api/v2/pokemon?limit=50'); const data = await respuesta.json(); // 🔥 filter() + map() encadenados const resultados = data.results .filter(pokemon => pokemon.name.includes(tipoBuscado)) // Filtra .map(pokemon => pokemon.name.toUpperCase()); // Transforma return resultados; }
🎯 Usamos .filter() para filtrar y .map() para transformar los nombres a mayúsculas.
