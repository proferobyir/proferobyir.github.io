📦 1. Estructura JSON (Objeto y Arreglo)

JSON = JavaScript Object Notation. Es la forma más común de intercambiar datos entre servidores y aplicaciones.

🔹 OBJETO JSON
{ "id": 101, "nombre": "Lina", "es_activo": true, "hobbies": ["futbol", "lectura"] }

📌 Llaves { } → representan un solo objeto.

🔸 ARREGLO JSON
[ { "id": 1, "nombre": "Ana" }, { "id": 2, "nombre": "Carlos" }, { "id": 3, "nombre": "Elena" } ]

📌 Corchetes [ ] → lista de objetos.

✅ Clave: Todo JSON válido usa comillas dobles "clave": valor.

📁 2. Async/Await · Datos locales

Simulamos una fuente de datos con un arreglo de 5 Pokémon locales.
Se usa Promise + setTimeout para imitar latencia.

// Datos locales (5 registros) const pokemonsLocal = [ { id: 1, nombre: "Pikachu", tipo: "eléctrico" }, { id: 2, nombre: "Charmander", tipo: "fuego" }, { id: 3, nombre: "Squirtle", tipo: "agua" }, { id: 4, nombre: "Bulbasaur", tipo: "planta" }, { id: 5, nombre: "Jigglypuff", tipo: "hada" } ]; async function obtenerPokemonsLocal() { return new Promise(resolve => { setTimeout(() => resolve(pokemonsLocal), 800); }); }
⚡ Haz clic en el botón para cargar datos locales.

🌐 3. API Externa · JSONPlaceholder

Consumimos JSONPlaceholder (API gratuita) con fetch y async/await. Mostramos 5 usuarios.

async function obtenerUsuariosAPI() { const respuesta = await fetch('https://jsonplaceholder.typicode.com/users'); const datos = await respuesta.json(); return datos.slice(0,5); }
🌍 Presiona para consultar API JSONPlaceholder.

⚡ 4. Pokémon API · Datos reales

Consumimos PokéAPI para obtener información de 3 Pokémon: bulbasaur, charmander, squirtle.

async function obtenerPokemonAPI(nombre) { const res = await fetch(`https://pokeapi.co/api/v2/pokemon/${nombre}`); const data = await res.json(); return { nombre: data.name, id: data.id, sprite: data.sprites.front_default }; }
✨ Selecciona un Pokémon o "Ver los 3" para llamar la API asíncrona.

🚀 5. Asincronía avanzada: Promise.all / concurrencia

Llamamos a dos APIs diferentes en simultáneo usando Promise.all para optimizar tiempos. Ejemplo educativo: JSONPlaceholder + PokéAPI (Pokémon aleatorio).

async function cargarMultipleAPIs() { const [usuarios, pokemon] = await Promise.all([ fetch('https://jsonplaceholder.typicode.com/users/1').then(r => r.json()), fetch('https://pokeapi.co/api/v2/pokemon/pikachu').then(r => r.json()) ]); return { usuario: usuarios.name, pokemon: pokemon.name }; }
🔄 Muestra el resultado de dos peticiones simultáneas (usuario + pikachu).

📖 ¿Qué significa "asíncrono"?

Las funciones asíncronas permiten que el código no se bloquee mientras espera una respuesta (de una API, archivo, base de datos). async convierte una función para que devuelva una promesa, y await espera la respuesta sin congelar la interfaz.

// Ejemplo estructura async function obtenerDatos() { try { const respuesta = await fetch('url'); const datos = await respuesta.json(); console.log(datos); } catch(error) { console.error('Error:', error); } }

💡 Consejo: Usa try/catch para manejar errores de red o endpoints caídos.
