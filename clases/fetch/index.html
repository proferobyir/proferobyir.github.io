📦 1. Estructura JSON que devuelve PokéAPI

Cuando consultamos https://pokeapi.co/api/v2/pokemon/pikachu, obtenemos un objeto con mucha información. Uno de los campos clave es sprites.front_default, que contiene la URL de la imagen.

{ "id": 25, "name": "pikachu", "sprites": { "front_default": "https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/25.png", "back_default": "https://raw.githubusercontent.com/.../back.png", "other": { ... } }, "types": [ { "slot": 1, "type": { "name": "electric" } } ], "height": 4, "weight": 60 }
🎯 sprites.front_default → Imagen frontal
📊 types → Tipos del Pokémon
🔢 id → Número nacional

Arreglo JSON local con 5 Pokémon y sus sprites. Usamos setTimeout para simular latencia y Promise.

const pokemonesLocal = [ { nombre: "Pikachu", tipo: "eléctrico", imagen: "https://raw.githubusercontent.com/.../25.png" }, { nombre: "Charmander", tipo: "fuego", imagen: "https://raw.githubusercontent.com/.../4.png" }, // ... 3 más ]; async function obtenerLocal() { return new Promise(resolve => { setTimeout(() => resolve(pokemonesLocal), 700); }); }

🎣 3. Async/Await + PokéAPI (Individual)

Función asíncrona que consume la API y extrae nombre, imagen y tipos.

async function obtenerPokemon(nombre) { const respuesta = await fetch(`https://pokeapi.co/api/v2/pokemon/${nombre}`); const data = await respuesta.json(); return { nombre: data.name, imagen: data.sprites.front_default, tipos: data.types.map(t => t.type.name).join(', ') }; }
🌟 Selecciona un Pokémon y verás su imagen desde la PokéAPI en tiempo real.

⚡ 4. Promise.all (Múltiples Pokémon)

Cargamos varios Pokémon en paralelo usando Promise.all. Ideal para optimizar tiempos.

const nombres = ["pikachu", "charizard", "mewtwo", "eevee", "snorlax"]; async function cargarVariosPokemon() { const promesas = nombres.map(n => fetch(`https://pokeapi.co/api/v2/pokemon/${n}`).then(r => r.json()) ); return await Promise.all(promesas); }
🐉 Muestra una galería con sprites de 5 Pokémon distintos.

🔍 5. Buscador de Pokémon (Asíncrono con manejo de errores)

Escribe el nombre de un Pokémon y al hacer clic se buscará su sprite y datos. Incluye try/catch para manejar Pokémon inexistentes.

async function buscarPokemon(nombre) { try { const res = await fetch(`https://pokeapi.co/api/v2/pokemon/${nombre.toLowerCase()}`); if (!res.ok) throw new Error("Pokémon no encontrado 😢"); const data = await res.json(); mostrarDatos(data); } catch(error) { mostrarError(error.message); } }
💡 Prueba escribiendo: pikachu, mew, gengar, lucario...

✨ 6. Detalle completo + Imagen HD (Shiny opcional)

Además del sprite normal, podemos mostrar la imagen shiny (variante de color) y estadísticas base.

async function obtenerDetalles(nombre) { const res = await fetch(`https://pokeapi.co/api/v2/pokemon/${nombre}`); const data = await res.json(); return { nombre: data.name, spriteNormal: data.sprites.front_default, spriteShiny: data.sprites.front_shiny, stats: data.stats.map(s => `${s.stat.name}: ${s.base_stat}`).join(' · ') }; }
🎯 Haz clic y mira la imagen normal, shiny y estadísticas del Pokémon.
