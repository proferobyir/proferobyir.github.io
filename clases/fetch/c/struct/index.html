🔍 Anatomía de un programa en C

Un programa en C sigue esta estructura básica:

#include
// función principal
int main() {
    printf("Hola mundo");
    return 0;
}
✅ Directivas #include: Librerías.
✅ main(): Punto de entrada.
✅ return 0: Indica final exitoso.

📦 Ejemplos de sintaxis clave

int edad = 21;
float altura = 1.75;
char inicial = 'A';
📌 Variables: tipos primitivos int, float, char.
if (edad >= 18) {
  printf("Mayor");
} else {
  printf("Menor");
}
⚡ Condicional if-else y estructura de bloques { }.
for(int i=0;i<5;i++){
  printf("%d ",i);
}
🔄 Bucle for típico, inicialización, condición e incremento.
int suma(int a, int b) {
  return a + b;
}
📚 Funciones: tipo de retorno, parámetros, return.

🧠 Buena práctica: comentarios y estructura

/* Programa: calcula el área */
#include

int main() {
    int base = 10, altura = 5;
    int area = base * altura;
    printf("Área: %d", area);
    return 0;
}
💡 Los comentarios /* */ y // documentan el código.
✨ El retorno de main suele ser 0 si todo funciona.
📌 Punteros (idea básica)
int x = 5; int *ptr = &x;
ptr guarda dirección de memoria.
📌 Arreglos
int nums[3] = {1,2,3};
Indexados desde 0.
🧪 C es estructurado: las funciones, bucles y condiciones organizan el flujo.
