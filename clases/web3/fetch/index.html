⚡ map() + CSS Grid (2 columnas) + PokéAPI

Aprendiendo Async/Await, .map() y CSS Grid para organizar Pokémon en 2 columnas

📊 ESQUEMA DE FUNCIONAMIENTO: .map() + CSS GRID

💡 ¿Por qué ya no flota el div? Se eliminó cualquier propiedad float o mala posición. Ahora el Grid actúa como contenedor bloque con display: grid y clear: both. Cada tarjeta respeta el flujo del Grid.

📁 1. .map() + Grid (Datos locales)

Transformamos un arreglo JSON de 5 Pokémon en una cuadrícula de 2 columnas usando CSS Grid.

// 🔥 FUNCIÓN mostrarGridLocal():
// 1. Toma array pokemonLocalData (objetos con nombre, imagen, tipo)
// 2. .map() → itera cada pokemon y retorna template HTML string
// 3. .join('') → une todos los strings en uno solo
// 4. Inyecta dentro de div con clase 'pokemon-grid-2cols'
// 5. CSS Grid: grid-template-columns: 1fr 1fr → organiza automático
⚡ Haz clic para ver cómo .map() + Grid 1fr 1fr organizan los Pokémon.

🌐 2. PokéAPI + map() + Grid 2 columnas

Consumimos PokéAPI con async/await, transformamos con .map() y mostramos en 2 columnas.

// FUNCIÓN mostrarGridAPI (async):
// 1. Array nombres = ["pikachu","charizard"...]
// 2. .map(nombre => fetch(...).then(res => res.json())) → promesas
// 3. Promise.all(promesas) → espera TODAS las respuestas en paralelo
// 4. .map() sobre resultados → genera HTML con nombre, tipo, imagen
// 5. join() y se inyecta en Grid 2 columnas
🌟 .map() genera el HTML, CSS Grid organiza en 2 columnas.

🔄 3. Grid 1fr 1fr vs Flexbox

Con Grid forzamos EXACTAMENTE 2 columnas. Con Flexbox las tarjetas se ajustan según el ancho.

/* 🔥 CSS Grid: columnas fijas 1fr 1fr -> siempre dos */ .pokemon-grid-2cols { display: grid; grid-template-columns: 1fr 1fr; }
/* 🔄 Flexbox: flujo flexible con wrap */ .flex-wrap { display: flex; flex-wrap: wrap; gap:1rem; }
💡 Muestra la diferencia entre Grid (2 columnas fijas) y Flexbox (responsive fluido).

📱 4. Grid Responsivo con Media Query

Usamos media query para que en móvil las tarjetas se muestren en 1 columna y en desktop en 2 columnas.

/* CSS responsivo: pantallas ≤640px cambia a 1fr */ @media (max-width: 640px) { .pokemon-grid-2cols { grid-template-columns: 1fr; } }
🖥️ Redimensiona tu navegador: en escritorio verás 2 columnas, en móvil 1 columna.

🎨 5. Grid avanzado: tarjetas con imágenes más grandes

Usamos Grid de 2 columnas pero con tarjetas más vistosas (imagen de 100px).

// La misma lógica .map() + join() pero con estilos mejorados .pokemon-card-grande { padding:1.2rem; img { width:100px; } }
🎯 Grid 1fr 1fr con tarjetas más grandes y detalles adicionales.
