⚡ map() + CSS Grid + PokéAPI

Sin floats, sin capas flotantes. Aprendiendo map, fetch y promesas con diseño limpio en Grid.

📚 Anatomía de funciones clave: .map() , fetch() y Promesas

🔁 .map() → Transforma arrays

// .map() recorre cada elemento y devuelve NUEVO array
const original = [1, 2, 3];
const doble = original.map( (numero) => numero * 2 );
// Resultado: [2, 4, 6]
┌─────────────────────────────────────────┐
│ arrayOriginal ──.map( callback )──► nuevoArray │
│ │
│ callback recibe: (elemento, índice, array)
│ → siempre retorna el elemento transformado │
└─────────────────────────────────────────┘
✨ Se usa para convertir datos en HTML, objetos, etc.

🌐 fetch() + Promesas

// fetch devuelve una PROMESA (estado pendiente)
fetch('https://api.ejemplo.com')
  .then( respuesta => respuesta.json() )
  .then( datos => console.log(datos) )
  .catch( error => manejarError(error) );
┌─────────┐ ┌──────────┐ ┌─────────────┐
│ fetch() │ ──► │ Promesa │ ──► │ Respuesta │
│ (petición) │ pendiente│ │ (resuelta) │
└─────────┘ └──────────┘ └─────────────┘
│ │ │
▼ ▼ ▼
[ .then() ] [ .catch() ] [ .finally() ]
✅ Las promesas evitan el "callback hell" y manejan asincronía.

⚡ map() + Promise.all (múltiples fetch)

// 1. Arreglo de urls o nombres
const urls = ['pikachu', 'charizard'];
// 2. .map() crea array de promesas
const promesas = urls.map( nombre =>
  fetch(`https://pokeapi.co/api/v2/pokemon/${nombre}`).then(res => res.json())
);
// 3. Promise.all espera TODAS
const resultados = await Promise.all(promesas);
[ url1, url2, url3 ]
│
▼
.map( fetch ) → [ promesa1, promesa2, promesa3 ]
│
▼
Promise.all( [promesa1, promesa2, promesa3] )
│
▼
[ data1, data2, data3 ] ← ¡todo en paralelo!
🚀 Ideal para cargar múltiples recursos simultáneamente.
💡 Resumen visual: map() transforma → fetch() lanza petición asíncrona → Promesa gestiona el resultado → ¡sin floats, sin superposiciones!

📁 1. .map() + Grid (local)

Transformamos datos locales en HTML con .map() y CSS Grid (2 columnas fijas).

// Datos locales
const pokemones = [ {id:25, nombre:"Pikachu"}, ... ];
// .map() genera tarjetas
const tarjetas = pokemones.map(p => `<div class="pokemon-card">...</div>`).join('');
// Grid 1fr 1fr aplica 2 columnas automáticas

🌐 2. PokéAPI + map() + Grid

.map() + Promise.all + fetch() → Grid organiza sin floats.

const nombres = ["pikachu","charizard","mewtwo"];
const promesas = nombres.map(n => fetch(...).then(r=>r.json()));
const data = await Promise.all(promesas);

🔄 3. Grid 1fr 1fr vs Flexbox

Comparativa: Grid fuerza 2 columnas exactas, Flex se adapta.

.grid-2cols { display: grid; grid-template-columns: 1fr 1fr; }
.flex-wrap { display: flex; flex-wrap: wrap; gap:1rem; }
Comparativa clara sin floats.
