Node.js & NPM: El Viaje del Flujo de Datos

Guía didáctica interactiva adaptada al entorno de Windows.

FASE 1

Inicialización Local: npm init

Al iniciar un proyecto, NPM trabaja de manera 100% interna en tu computadora. Procesa los datos en la memoria RAM y escribe directamente el archivo de configuración en tu disco duro sin salir a Internet.

PS C:\Users\Profe\Desktop> mkdir mi-app
PS C:\Users\Profe\Desktop> cd mi-app
PS C:\Users\Profe\Desktop\mi-app> npm init -y
# Genera el archivo de configuración en el disco C:\...

Resultado en el disco duro (package.json):

{
  "name": "mi-app",
  "version": "1.0.0",
  "main": "index.js",
  "dependencies": {}
}
FASE 2

Instalación y Petición Externa: npm install

Aquí inicia el verdadero viaje externo. Windows abre un puerto de red y NPM realiza una petición HTTPS segura al registro global para resolver y traer las dependencias requeridas.

PS C:\Users\Profe\Desktop\mi-app> npm install express
# 1. Consulta externa a https://registry.npmjs.org
# 2. Descarga de paquetes comprimidos (.tgz) vía CDN
# 3. Extracción automática en la carpeta local 'node_modules'
