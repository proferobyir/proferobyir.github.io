Variables y Tipos de Datos
let, const, var y los tipos básicos
Las variables son espacios en memoria donde guardamos información. En ECMAScript usamos let para valores que cambian, const para valores fijos, y var (obsoleto).
EJEMPLO // ============================================= // 1. DECLARACIÓN DE VARIABLES // ============================================= // let: para valores que cambian let nombre = 'Juan'; nombre = 'Carlos'; // ✅ Se puede cambiar // const: para valores fijos const PI = 3.1416; // PI = 3.14; ❌ Error: no se puede reasignar // Tipos de datos const texto = 'Hola Mundo'; // String const numero = 42; // Number const verdadero = true; // Boolean const nulo = null; // Null const indefinido = undefined; // Undefined
✏️ Ejercicio:

Crea variables para guardar el nombre, edad y ciudad de un usuario.

2
Funciones
Bloques de código reutilizables
Las funciones son bloques de código que podemos reutilizar. Reciben parámetros y pueden devolver valores.
EJEMPLO // ============================================= // 2. FUNCIONES // ============================================= // Función tradicional function saludar(nombre) { return 'Hola ' + nombre; } // Función con múltiples parámetros function sumar(a, b) { return a + b; } // Llamar a las funciones const mensaje = saludar('Juan'); // 'Hola Juan' const resultado = sumar(5, 3); // 8 // Función sin return (devuelve undefined) function mostrarMensaje(texto) { console.log(texto); // No tiene return }
✏️ Ejercicio:

Crea una función que reciba el precio de un producto y devuelva el precio con IVA (16%).

3
Arrays
Listas de datos
Los arrays son listas ordenadas de elementos. Pueden contener cualquier tipo de dato.
EJEMPLO // ============================================= // 3. ARRAYS // ============================================= // Crear un array const frutas = ['Manzana', 'Banana', 'Naranja']; // Acceder a elementos (índice 0) console.log(frutas[0]); // 'Manzana' // Agregar elementos frutas.push('Uva'); // Agrega al final frutas.unshift('Fresa'); // Agrega al inicio // Eliminar elementos frutas.pop(); // Elimina el último frutas.shift(); // Elimina el primero // Recorrer arrays con for for (let i = 0; i < frutas.length; i++) { console.log(frutas[i]); } // Recorrer con for...of for (const fruta of frutas) { console.log(fruta); } // Arrays de objetos (¡MUY IMPORTANTE!) const productos = [ { id: 1, nombre: 'Laptop', precio: 1000 }, { id: 2, nombre: 'Teléfono', precio: 500 }, { id: 3, nombre: 'Tablet', precio: 300 } ];
✏️ Ejercicio:

Crea un array con 5 productos (nombre y precio). Recórrelo y muestra cada producto.

4
Objetos
Estructuras de datos con propiedades
Los objetos son estructuras de datos que agrupan información en pares clave-valor.
EJEMPLO // ============================================= // 4. OBJETOS // ============================================= // Crear un objeto const persona = { nombre: 'Juan', edad: 25, ciudad: 'Caracas', saludar: function() { return 'Hola, soy ' + this.nombre; } }; // Acceder a propiedades console.log(persona.nombre); // 'Juan' console.log(persona['edad']); // 25 // Modificar propiedades persona.edad = 26; // Agregar propiedades persona.profesion = 'Ingeniero'; // Destructuring (MUY IMPORTANTE) const { nombre, edad } = persona; console.log(nombre); // 'Juan' // Objeto dentro de otro objeto const producto = { id: 1, nombre: 'Laptop', caracteristicas: { marca: 'Dell', ram: '16GB', almacenamiento: '512GB' } }; console.log(producto.caracteristicas.marca); // 'Dell'
✏️ Ejercicio:

Crea un objeto producto con nombre, precio, categoría y una función que calcule el precio con descuento.

5
Template Literals
Interpolación de cadenas con backticks
Los template literals permiten insertar variables directamente en strings usando ${} y el carácter de backtick `.
EJEMPLO // ============================================= // 5. TEMPLATE LITERALS // ============================================= const nombre = 'Juan'; const edad = 25; // Sin template literals (concatenación) const mensaje1 = 'Hola ' + nombre + ', tienes ' + edad + ' años'; // Con template literals (MUCHO MÁS FÁCIL) const mensaje2 = `Hola ${nombre}, tienes ${edad} años`; // Template literals para HTML (¡IMPORTANTE!) const producto = { nombre: 'Laptop', precio: 1000 }; // Generar HTML dinámico const html = ` <div class="producto"> <h3>${producto.nombre}</h3> <p>Precio: $${producto.precio}</p> </div> `; // Multi-línea sin problemas const multilinea = ` Esta es una cadena de múltiples líneas sin necesidad de \n `;
✏️ Ejercicio:

Usa template literals para crear HTML de una tarjeta de producto con nombre, precio e imagen.

6
Arrow Functions
Funciones flecha (sintaxis moderna)
Las arrow functions son una forma más corta de escribir funciones. Usan la sintaxis () => {}.
EJEMPLO // ============================================= // 6. ARROW FUNCTIONS // ============================================= // Función tradicional function sumar(a, b) { return a + b; } // Arrow function (versión larga) const sumar2 = (a, b) => { return a + b; }; // Arrow function (versión corta - sin llaves ni return) const sumar3 = (a, b) => a + b; // Un solo parámetro (sin paréntesis) const doble = x => x * 2; // Sin parámetros const saludar = () => 'Hola Mundo'; // Arrow function con objeto (¡CUIDADO con el return!) const crearProducto = (nombre, precio) => ({ nombre: nombre, precio: precio }); // Arrow functions en arrays (MUY COMÚN) const numeros = [1, 2, 3, 4, 5]; const dobles = numeros.map(n => n * 2); // [2,4,6,8,10]
✏️ Ejercicio:

Convierte esta función tradicional a arrow function:
function calcularDescuento(precio) { return precio * 0.9; }

7
map, filter, reduce
Transformar, filtrar y acumular arrays
Estos métodos son fundamentales para trabajar con arrays. map() transforma, filter() filtra y reduce() acumula.
EJEMPLO // ============================================= // 7. map, filter, reduce // ============================================= const productos = [ { id: 1, nombre: 'Laptop', precio: 1000, categoria: 'Electrónicos' }, { id: 2, nombre: 'Teléfono', precio: 500, categoria: 'Electrónicos' }, { id: 3, nombre: 'Camisa', precio: 50, categoria: 'Ropa' }, { id: 4, nombre: 'Pantalón', precio: 80, categoria: 'Ropa' }, { id: 5, nombre: 'Smartwatch', precio: 200, categoria: 'Electrónicos' } ]; // ============================================= // map() - Transforma cada elemento // ============================================= // Obtener solo los nombres const nombres = productos.map(p => p.nombre); // ['Laptop', 'Teléfono', 'Camisa', 'Pantalón', 'Smartwatch'] // Crear HTML con map() const htmlProductos = productos.map(p => ` <div class="producto"> <h3>${p.nombre}</h3> <p>$${p.precio}</p> </div> `).join(''); // ============================================= // filter() - Filtra elementos // ============================================= // Productos electrónicos const electronicos = productos.filter(p => p.categoria === 'Electrónicos'); // Productos con precio > 100 const productosCaros = productos.filter(p => p.precio > 100); // ============================================= // reduce() - Acumula valores // ============================================= // Sumar todos los precios const total = productos.reduce((acumulador, p) => acumulador + p.precio, 0); // Combinar map + filter // Obtener nombres de productos caros const nombresCaros = productos .filter(p => p.precio > 100) .map(p => p.nombre);
✏️ Ejercicio:

Dado un array de productos, usa filter para obtener solo los de categoría "Ropa" y map para obtener solo sus nombres.

8
Async / Await
Peticiones asíncronas a servidores
Async/Await es la forma moderna de hacer peticiones a servidores. Permite que el código espere a que llegue la respuesta antes de continuar.
EJEMPLO // ============================================= // 8. ASYNC/AWAIT // ============================================= // ============================================= // fetch() - Hace peticiones HTTP // ============================================= // fetch con Promesas (manera antigua) fetch('http://localhost:3000/api/productos') .then(response => response.json()) .then(data => console.log(data)) .catch(error => console.error(error)); // ============================================= // ASYNC/AWAIT (manera moderna) // ============================================= // Definir función asíncrona async function obtenerProductos() { try { // Esperar a que la petición termine const respuesta = await fetch('http://localhost:3000/api/productos'); // Verificar si la respuesta es correcta if (!respuesta.ok) { throw new Error('Error al obtener productos'); } // Convertir la respuesta a JSON const productos = await respuesta.json(); // Usar los datos console.log(productos); return productos; } catch (error) { // Manejar errores console.error('Error:', error); // Mostrar mensaje al usuario document.getElementById('error').textContent = 'Error al cargar datos'; } } // ============================================= // POST con Async/Await // ============================================= async function crearProducto() { const nuevoProducto = { nombre: 'Producto Nuevo', precio: 100 }; try { const respuesta = await fetch('http://localhost:3000/api/productos', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(nuevoProducto) }); const data = await respuesta.json(); console.log('Producto creado:', data); } catch (error) { console.error('Error al crear:', error); } } // ============================================= // Mostrar datos en HTML // ============================================= async function mostrarProductos() { try { const productos = await obtenerProductos(); const container = document.getElementById('productos'); // Generar HTML con template literals container.innerHTML = productos.map(producto => ` <div class="card"> <h3>${producto.nombre}</h3> <p>$${producto.precio}</p> </div> `).join(''); } catch (error) { console.error('Error:', error); } } // Llamar a la función mostrarProductos();
✏️ Ejercicio:

Crea una función async que obtenga productos de una API y los muestre en el HTML.
