Node.js & NPM: El Viaje del Flujo de Datos

Interactúa con las fases del proyecto para visualizar cómo se comporta la información de manera interna y externa.

FASE 1

Inicialización Local: npm init

Al ejecutar la inicialización de un paquete, el asistente compila tus datos de configuración directamente en un archivo local sin salir de tu computadora.

yir@pc:~/mi-app$ npm init -y
# Crea instantáneamente el archivo de configuración base...

{
  "name": "mi-app",
  "version": "1.0.0",
  "main": "index.js",
  "dependencies": {}
}
FASE 2

Instalación y Petición: npm install express

Aquí inicia el viaje de datos. NPM genera una petición HTTPS hacia el registro público buscando los metadatos y resolviendo el árbol de dependencias necesarias.

yir@pc:~/mi-app$ npm install express
# 1. Consulta al registro externo (https://registry.npmjs.org)
# 2. Descarga de Tarballs (.tgz) mediante la red de distribución (CDN)
# 3. Extracción automática en la carpeta local 'node_modules'
