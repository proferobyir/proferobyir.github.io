PowerShell Commands

Guía rápida para archivos y carpetas
📁 Files & Folders ✔️ PowerShell 7+
📋

Comandos esenciales para gestionar archivos y carpetas en PowerShell. Organizados por categoría con ejemplos prácticos.

New-Item Remove-Item Copy-Item
⚠️
¡Cuidado con Remove-Item -Recurse -Force! Elimina carpetas con todo su contenido sin confirmación. Siempre prueba con -WhatIf primero.
📄

Archivos

Files
# Crear archivo vacío New-Item -Path "archivo.txt" -ItemType File
Crea un archivo vacío en la ruta especificada.
# Crear con contenido "Hola mundo" | Out-File archivo.txt
Escribe contenido directamente en el archivo.
# Borrar archivo Remove-Item -Path "archivo.txt"
Elimina el archivo de forma permanente.
# Renombrar Rename-Item -Path "viejo.txt" -NewName "nuevo.txt"
Cambia el nombre del archivo.
# Listar archivos Get-ChildItem -Path "C:\ruta" -File
Muestra solo los archivos de la carpeta.
# Listar con filtro Get-ChildItem -Path "C:\ruta" -Filter "*.txt" -File
Filtra archivos por extensión.
📂

Carpetas

Folders
# Crear carpeta New-Item -Path "C:\nueva_carpeta" -ItemType Directory
Crea una nueva carpeta (también funciona mkdir).
# Borrar carpeta vacía Remove-Item -Path "C:\carpeta" -ItemType Directory
Elimina solo si la carpeta está vacía.
# Borrar carpeta con contenido Remove-Item -Path "C:\carpeta" -Recurse -Force
⚠️ Elimina todo sin preguntar.
# Borrar con confirmación Remove-Item -Path "C:\carpeta" -Recurse -Confirm
Pide confirmación antes de cada eliminación.
# Renombrar carpeta Rename-Item -Path "C:\vieja" -NewName "nueva"
Cambia el nombre de la carpeta.
# Listar carpetas Get-ChildItem -Path "C:\ruta" -Directory
Muestra solo los directorios.
# Listar recursivo Get-ChildItem -Path "C:\ruta" -Directory -Recurse
Muestra todas las subcarpetas.
📋

Copiar Carpetas

Copy
# Copiar carpeta completa Copy-Item -Path "C:\origen\carpeta" -Destination "C:\destino\" -Recurse
Copia la carpeta y todo su contenido.
# Copiar y sobrescribir Copy-Item -Path "C:\origen\carpeta" -Destination "C:\destino\" -Recurse -Force
Sobrescribe archivos existentes.
# Copiar solo el contenido Copy-Item -Path "C:\origen\carpeta\*" -Destination "C:\destino\" -Recurse
Copia los archivos dentro de la carpeta, no la carpeta en sí.
# Copiar con estructura Copy-Item -Path "C:\origen\carpeta" -Destination "C:\destino\carpeta" -Recurse
Mantiene la misma estructura de carpetas.
🔧

Comandos Útiles

Extras
# Mover archivo/carpeta Move-Item -Path "origen" -Destination "destino"
Mueve (corta y pega) el elemento.
# Ver contenido de archivo Get-Content archivo.txt
Muestra el contenido del archivo en consola.
# Ver propiedades Get-Item archivo.txt | Format-List *
Muestra todas las propiedades del elemento.
# Buscar archivos Get-ChildItem -Recurse -Filter "*.pdf"
Busca archivos PDF en toda la estructura.
# Contar archivos (Get-ChildItem -File).Count
Cuenta cuántos archivos hay en la carpeta actual.
# Simular antes de ejecutar Remove-Item -Path "carpeta" -Recurse -WhatIf
¡Seguridad! Muestra qué se eliminaría sin hacerlo.
💻 Ejemplo práctico completo
# 1. Crear estructura de proyecto mkdir C:\Proyecto\Docs New-Item C:\Proyecto\Docs\README.md -Value "# Proyecto" # 2. Listar todo recursivamente Get-ChildItem C:\Proyecto -Recurse # 3. Copiar carpeta Docs a Backup Copy-Item C:\Proyecto\Docs D:\Backup\ -Recurse -Force # 4. Renombrar archivo Rename-Item C:\Proyecto\Docs\README.md -NewName "INICIO.md" # 5. Simular eliminación del proyecto Remove-Item C:\Proyecto -Recurse -WhatIf # 6. Eliminar definitivamente (¡cuidado!) Remove-Item C:\Proyecto -Recurse -Force
