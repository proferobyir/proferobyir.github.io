🧠 🌐 Modelo Cliente-Servidor | ⚡ Async/Await | 🚀 Consumo de API

🖥️ Cliente ↔ 🌍 Servidor

Cliente (navegador) envía peticiones HTTP.
Servidor procesa y responde (JSON/XML).
La comunicación es asíncrona para no bloquear la UI.

⏳ Funciones Asíncronas

async/await + Promises. Una función async siempre retorna una Promise. await espera la resolución sin bloquear el hilo principal. Ideal para peticiones a APIs.

📡 Consumo de API (REST)

Usamos fetch() o librerías. fetch(url) devuelve Promise. Luego response.json() extrae datos. Patrón: try/catch para manejo de errores de red/servidor.

GET POST JSON
💡 Consumo de API típico: const res = await fetch('https://api.ejemplo.com/datos'); const data = await res.json();
