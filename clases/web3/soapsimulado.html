📖 ¿Qué es SOAP?

📦 Envelope (Sobre)

Elemento raíz que contiene todo el mensaje SOAP. Es obligatorio.

📋 Header (Cabecera)

Opcional. Contiene metadatos como autenticación o transacciones.

🎯 Body (Cuerpo)

Obligatorio. Contiene los datos de la llamada o respuesta.

⚡ HTTP POST

SOAP usa HTTP como transporte, normalmente con método POST.
