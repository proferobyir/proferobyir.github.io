💡 ¿Qué es SOAP?
SOAP (Simple Object Access Protocol) es un protocolo que usa XML para comunicar aplicaciones. Cada mensaje tiene un Envelope (sobre), Header (opcional) y Body (cuerpo).

📨 Mensajes SOAP

📖 Anatomía de un Mensaje SOAP

📦 Envelope

El sobre que contiene todo el mensaje. Es el elemento raíz obligatorio.

📋 Header

Opcional. Contiene metadatos como autenticación.

🎯 Body

Obligatorio. Contiene los datos de la llamada o respuesta.
