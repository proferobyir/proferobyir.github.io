Planificación Académica Semestral

Guía de contenido estructurado por semanas para las asignaturas de Tecnología y Desarrollo Web.

Redes y Web 20 Semanas

Introducción a la infraestructura de internet y bases del desarrollo web estático.

Módulo 1: Redes (Semanas 1-10)
Sem 1
Arquitectura Cliente-Servidor y tipos de redes (LAN, WAN).
Sem 2-3
Direccionamiento físico y lógico: Dirección MAC e IP (v4/v6).
Sem 4-5
Máscaras de subred, Gateways y funcionamiento de DNS.
Sem 6-7
El Modelo OSI completo (Análisis detallado capa por capa).
Sem 8-9
Protocolos esenciales: TCP vs UDP, HTTP, HTTPS, FTP y SSH.
Sem 10
Evaluación práctica de Redes y conectividad.
Módulo 2: Web 1 (Semanas 11-20)
Sem 11-13
Anatomía de URLs, ciclo HTTP, VS Code + Live Server y HTML5 Semántico.
Sem 14-15
CSS3 básico: Selectores, Cascada, Especificidad y Box Model.
Sem 16-17
Maquetación moderna con Flexbox y diseño responsivo básico.
Sem 18-19
Control de versiones con Git/GitHub y despliegue en GitHub Pages.
Sem 20
Defensa del Proyecto Final (Sitio estático responsivo en producción).

Desarrollo Web 2 20 Semanas

Enfoque completo en la lógica, interactividad y asincronismo con JavaScript Nativo.

Bloque 1: Fundamentos (Sem 1-7)
Sem 1-3
Variables (let, const), tipos de datos, condicionales y bucles.
Sem 4-5
Funciones flecha, Arrays, Objetos literales y Destructuring.
Sem 6-7
Métodos avanzados de Arrays (.map, .filter, .reduce).
Bloque 2: DOM y Navegador (Sem 8-13)
Sem 8-10
Manipulación del DOM, eventos de usuario (click, keyboard) y clases CSS.
Sem 11-12
Validación de formularios y almacenamiento con localStorage.
Sem 13
Primer Proyecto Integrador: To-Do List interactiva.
Bloque 3: Asincronismo (Sem 14-20)
Sem 14-15
Manejo de asincronía: Callbacks y Promesas.
Sem 16-17
Consumo de APIs con Fetch, formato JSON y Async/Await con Try/Catch.
Sem 18-19
Renderizado dinámico de datos de APIs y técnicas de depuración.
Sem 20
Entrega de Proyecto Final (Aplicación interactiva basada en API pública).

Desarrollo Web 3 20 Semanas

Entorno profesional, Arquitectura de Componentes (React) e Introducción al Full Stack.

Bloque 1: Entorno Profesional (Sem 1-4)
Sem 1-2
Node.js, terminal de comandos y gestión de dependencias con npm.
Sem 3-4
POO en JS, Módulos ES6 (import/export) y estructuras limpias de carpetas.
Bloque 2: Frontend con React (Sem 5-12)
Sem 5-6
Arquitectura SPA, Vite, sintaxis JSX y diseño de Componentes con Props.
Sem 7-9
Manejo del estado con useState, renderizado condicional y hook useEffect.
Sem 10-12
Consumo de APIs en React, formularios controlados e introducción a Zustand.
Bloque 3: Backend y Base de Datos (Sem 13-20)
Sem 13-14
Creación de servidores HTTP con Express y diseño de APIs REST (Postman).
Sem 15-16
Bases de datos relacionales SQL, Normalización y consultas CRUD básicas.
Sem 17-18
Conexión de React con el Backend. Seguridad básica (CORS, variables .env).
Sem 19-20
Despliegue del ecosistema Full Stack (Vercel + Railway/Render) y defensa.

Avances Tecnológicos 10 Semanas

Laboratorio práctico de innovación: Automatización Low-Code, Videojuegos e IA.

Bloque 1: Automatización No-Code (Sem 1-3)
Sem 1
Introducción al ecosistema Low-Code/No-Code y su valor empresarial.
Sem 2-3
Automatización con n8n: Triggers, nodos y flujos reales (Email, Sheets, Telegram).
Bloque 2: Videojuegos con Unity (Sem 4-6)
Sem 4
Interfaz del motor Unity, jerarquía de objetos, escenas y componentes básicos.
Sem 5
Scripting básico en C#: Movimiento físico de objetos y detección de colisiones.
Sem 6
Desarrollo guiado de un mini-juego 2D/3D (Ej: Recolector de monedas).
Bloque 3: IA y Redes Neuronales (Sem 7-10)
Sem 7
Conceptos clave: Machine Learning, Deep Learning y la neurona artificial.
Sem 8
Estructura de Redes Neuronales: Capas (entrada, ocultas, salida), pesos y sesgos.
Sem 9
Práctica interactiva usando Google Teachable Machine o Keras en Google Colab.
Sem 10
Presentación y demostración de los proyectos de innovación de los alumnos.
