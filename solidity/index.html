De Cero a Programador de Blockchain

Una guía interactiva secuencial para aprender a pensar en código y dominar Solidity paso a paso.

Nivel 1: Los Datos Básicos

¿Dónde se guarda la información?

Para programar, necesitas "cajas" para guardar datos. Las llamamos Variables. En Solidity debes decirle a la caja qué tipo de dato va a guardar estrictamente. 

// Caja para números enteros
uint256 public precioCafe = 2;
// Caja para texto
string public mensaje = "¡Hola!";

Nivel 2: Tomar Decisiones

El control de flujo (If / Require)

Los programas toman decisiones. En Solidity usamos require(), que funciona como un filtro estricto: si no cumples la regla, detiene el programa y devuelve el dinero. 

// Si el pago es menor al costo, falla
require(pago >= precioCafe, "Invalido");

Nivel 3: El Molde (POO)

Estructurar un Objeto completo

En la Programación Orientada a Objetos juntamos las variables (atributos) y las acciones (funciones) dentro de un molde llamado Clase (o Contrato). 

contract MaquinaExpendedora {
    uint256 public stock; // Atributo

    function surtir(uint256 _cant) public {
        stock += _cant; // Acción
    }
}

Atributo -> Stock: 0 u.

Nivel 4: El Smart Contract

Contrato Inteligente en Red

Al darle a Desplegar, el molde sube a la Blockchain. El contrato ahora corre solo, es público y nadie lo puede alterar. 

contract Final {
    uint256 public stock = 3;
    
    function comprar() public payable {
        require(stock > 0, "Agotado");
        stock -= 1;
    }
}
