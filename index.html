🗺️ La Propuesta de Temario Práctico

Un recorrido diseñado para pasar de cero a crear flujos de trabajo automatizados y profesionales en solo 3 clases. Sin teoría aburrida, 100% basado en proyectos.

Clase 1

El "Anzuelo" (Tu primera automatización)

Proyecto: Captador de Leads Automatizado

Cómo funciona: Creas un formulario web sencillo usando el nodo nativo de n8n, sin herramientas externas. Cuando alguien lo llena, n8n procesa los datos, calcula algo automáticamente (como un puntaje o un saludo personalizado) y te envía una alerta instantánea.

Objetivo: Entender qué es un Trigger (Disparador), un nodo de datos (Edit Fields) y una Acción.
Captura y Procesamiento de Leads

Estructura del Workflow (Nodos)

1. TRIGGER

Webhook / Form Node

2. DATOS

Edit Fields (Set)

3. ACCIÓN

Discord / Telegram / Email

Clase 2

El conector del mundo real (Integración con IA y APIs)

Proyecto: Clasificador de correos con Inteligencia Artificial

Cómo funciona: Conectas un webhook que reciba un texto (o un nodo de Gmail) y se lo mandas a OpenAI (ChatGPT) para que analice el sentimiento (positivo/negativo). Si el resultado de la IA detecta que es un mensaje negativo, el flujo genera una respuesta automática de disculpa y te envía un aviso urgente.

Automatización Conectada con OpenAI

Estructura del Workflow (Nodos)

1. ENTRADA

Gmail / Webhook

2. INTELIGENCIA

OpenAI Node

3. CONDICIONAL

If / Switch Node

4. RESPUESTA

Gmail Draft / Slack

Clase 3

La Base de Datos y reportes

Proyecto: Sincronizador a Google Sheets o Notion

Cómo funciona: Aprenderás a guardar de forma estructurada y limpia todo el historial de eventos que pasa en tus flujos anteriores. Al final de la clase, programaremos un disparador de tiempo para extraer esos datos y generar un reporte consolidado automático directo a tu email corporativo.

Estructuración y Almacenamiento de Datos

Estructura del Workflow (Nodos)

1. RECOLECCIÓN

Google Sheets / Notion

2. CRONOGRAMA

Schedule Trigger (6:00 PM)

3. ENVÍO

Email / Digest Node
